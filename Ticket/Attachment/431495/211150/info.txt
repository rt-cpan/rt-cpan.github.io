Commit test for the path including multibyte character.
Both directory and file name contain 0x5c (back slash) and 0x7c 
(vertical bar) respectively when they were encoded in Shift_JIS.
マルチバイト文字をパスに含むファイルのコミットテストです。
「能」「申」はShift_JISで2バイト目に0x5cを含む、いわゆる「ダメ文字」です。
「芸」「ポ」はShift_JISで2バイト目に0x7c(|,パイプ)を含みます。
