Hi there,
 
I'd like to report a vulnerability in XML::Simple which relates to how it handles XML entities both internal and externally defined.  I believe this may affect more than simply XML::Simple although I haven't had a chance to create PoC for the implementations of XML parsers on which XML::Simple depends.  This relates to the following thread on the oss-security mailing list:
  • http://article.gmane.org/gmane.comp.security.oss.general/9421
Normally we try to practice co-ordinated discosure but the maintainer (Cc'd) has suggested that this would be right forum to get this resolved.
