Subject: [BUG AND PATCH] item undefined item
So I was writing some code to use the result of a whois query, however, Net::Whois::ARIN::Network was returning an error when the item was undefined rather then returning undef. Here is a patch and some code to demonstrate the bug/fix as well. Regards, Josh
Subject: Network-undef.patch
--- Network-orig.pm 2007-12-23 23:45:31.000000000 -0500 +++ Network.pm 2007-12-23 23:46:23.000000000 -0500 @@ -174,7 +174,7 @@ } } - croak "Undefined subroutine \&$AUTOLOAD called"; + return undef; } =head1 AUTHOR
