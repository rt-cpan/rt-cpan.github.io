Subject: patch for creating/using local persistent check daemon
assuming that the user has installed an rpm/dpkg/etc for the w3c validator check script, this patch starts a HTTP::Server::Simple::CGI and reads in the check script. The patch then provides WebService::Validator::HTML::W3C::Embedded as a super class of WebService::Validator::HTML::W3C, the super class just starts the local http server and calls the WebService::Validator::HTML::W3C->new method with the specified parameters.
