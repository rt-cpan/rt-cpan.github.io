Subject: I've fixed Math::VecStat to build under Perl 5.8
On ActivePerl 5.8 on Windows XP, Math::VecStat breaks on test 36 of Math-VecStat.t. This is because Perl 5.8 sorts slightly differently than perl 5.6.
