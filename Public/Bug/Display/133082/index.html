  Thu Jul 30 13:31:43 2020 ALEENA [...] cpan.org - Correspondence added
IF you were trying for something like a block quote, I think I found a solution. ----------- =pod =encoding utf8 =begin html <blockquote>This is a raw HTML blockquote.</blockquote> =end html =begin text This is could be a text quote. =end text =cut ----------- When the pod is rendered in a terminal emulator, the text block will be used; and if rendered on a site like meta::cpan, the html block will be used. I may be wrong and keeping two blocks for the same line may be a pain, but you MIGHT get what you want. However, meta::cpan has no css attached to <blockquote> and does not follow inline styles (at least not in the Pod Renderer). I don't know if this helps at all, and I hope this isn't overkill. Lady Aleena
