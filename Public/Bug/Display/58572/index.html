Subject: add distance method to count changed bits
Feature enhancement. hex_distance(n1, n2) measures the bit distance, as described on http://ixazon.dynip.com/~cmeclax/nilsimsa.html A build including this patch can be found in software.opensuse.org/search?q=perl-Digest-Nilsimsa
