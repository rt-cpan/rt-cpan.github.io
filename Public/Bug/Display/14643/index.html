Subject: Perlmonks Link Broken in Pod
Hi there. I was just browsing the docs for this module on Cpan Search: http://search.cpan.org/~ovid/Data-Record-0.01/lib/Data/Record.pm When I found that the link to the 'PerlMonks' thread doesn't work. It displays as this: http://perlmonks.org/index.pl?node_id=492002 But actually links to: http://search.cpan.org/perldoc?this%20thread Note that perlpod says: Or you can link to a web page: o "L<scheme:...>" Links to an absolute URL. For example, "L<http://www.perl.org/>". But note that there is no corre- sponding "L<text|scheme:...>" syntax, for various reasons. So you should just put the URL in the L<...> tag for this link, without any link text. (But I think you've got the bits either side of the vertical bar the wrong way round anyway!) Cheers. Smylers
