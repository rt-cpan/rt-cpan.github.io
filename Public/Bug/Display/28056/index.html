Hi, the Tcl::Tk requires snit package to be in Tcl/Tk installation for scrolled windows support. Snit is a part of 'tcllib' If you have ActiveTCL, use teapot to install snit. Or just download snit from tcllib repository (see http://wiki.tcl.tk/3963 and http://wiki.tcl.tk/1246) ActivePerl has snit within its tkkit.dll So Tcl::Tk should either go with snit embedded into distribution, or detect and report in Makefile.PL time.
