Subject: Additional package: File::Listing::apache
Attached is a patch to add a new File::Listing::apache package to the File::Listing module. The listing module should work for the standard Apache directory listing. There's also a new test file which is not enabled by default but can be run separately by using perl -Mblib t/base/apache-listing.t -f I hope this could go into the standard libwww distribution. If not, I would like to make a separate package. Regards, Slaven
