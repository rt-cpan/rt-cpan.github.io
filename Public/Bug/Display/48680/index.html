The dates used assume a format like 2009/08/12 or 2009/08/12T13:57:10 DateTime uses a slightly different format, i.e -- $ perl -MDateTime -le 'print DateTime->now->ymd' 2009-08-13 $ perl -MDateTime -le 'print DateTime->now->datetime' 2009-08-13T12:57:23 -- It is possible to use -- $ perl -MDateTime -le 'print DateTime->now->ymd("/")' 2009/08/13 -- but that doesn't help with '->datetime' Attached is a patch that copes with DateTime style dates and tests for them (copied existing tests and amended all the datestrings)
