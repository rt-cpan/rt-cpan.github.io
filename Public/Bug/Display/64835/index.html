As per subject. You have chomp( $line = <STDIN> ); which simply stops forever on a smoker machine. Typically such processes will be killed so your test coverage by cpantesters stays much lower than necessary. For such cases MakeMaker offers a prompt function that can be influenced by environment variables and default settings. Please use it. Thanks,
