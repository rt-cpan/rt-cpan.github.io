Hi Todd, thanks for the feedback. The IO-Compress-Zlib *distribution* is available on CPAN http://search.cpan.org/~pmqs/IO-Compress-Zlib-2.006/ Which doesn't exist is a perl *module* called IO::Compress::Zlib. The problem is down to the CPAN shell not knowing what to do with distributions where there isn't a module that matches the distribution name. I guess we are stuck with the behaviour in CPAN for the moment, so I'll redo the paragraph on the prerequisites. cheers Paul
