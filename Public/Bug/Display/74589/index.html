Compilation by GCC 4.7 with CFLAGS -O2 -g -pipe -Wall -Wp,-D_FORTIFY_SOURCE=2 produces various warning about mismatching printf formatting strings. Attached two patches silent the warnings by using more explicit coercion (first apply 0001-Fix-compiler-warnings.patch, then 0001-Fix-64-bit-warnings.patch).
