The following mini-program shows a regression in perl 5.18: @Class2::ISA = qw(Class1); sub Class1::AUTOLOAD { *{$Class1::AUTOLOAD} = sub { print "defined\n" }; print "autoloaded\n"; } Class2->Class2::SUPER::a; Class2->Class2::SUPER::a; Class2->a; Class2->a; In perl versions prior to 5.18.0 it prints: autoloaded defined autoloaded defined In 5.18.2 (and 5.18.0) it prints "autoloaded" 3 times in a row. And Class2->can('Class2::SUPER::a') returns undef instead of CODE, although the method gets re-defined every time, causing warnings "Subroutine SubClass::SUPER::method redefined" in my real code.
