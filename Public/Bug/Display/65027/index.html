> Looks like the format Weather Underground shows temperature in my > have changed recently. Attached is a (not great) patch for > getting temperature working again.
A little regression in your patch. You use the same regex for Â°C and Â°F so the same value is put in temperature_celsius and temperature_fahrenheit. Replace the regex "/(-?([?[0-9]*\.?[0-9]+))/i" by : * /(-?([?[0-9]*\.?[0-9]+))&deg;c/i) for celsius * /(-?([?[0-9]*\.?[0-9]+))&deg;f/i) for fahrenheit seems to work better (but I'm not sure that the "&deg;(c|f)" format is always used) See corrected patch file enclosed.
