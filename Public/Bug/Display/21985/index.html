Subject: Quiet warning in Manifest.t
Manifest.t removed the $warn initialization in catch_warning() causing a use of uninitialized value warning in one of the tests. I've put that initialization back. I've also changed a "cmp_ok( $foo, 'eq', $bar )" to the simpler "is( $foo, $bar )" and some minor whitespace nits.
