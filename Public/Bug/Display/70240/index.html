Wx::Scintilla uses a customized Build.PL that accepts additional options. Unfortunately, it doesn't accepts standard Module::Build options such as --installdirs On Ubuntu we get "--installdirs site" as a standard value CPAN.pm configuration option 'mbuildpl_arg' which is given to Build.PL. As a result the configure fails.
