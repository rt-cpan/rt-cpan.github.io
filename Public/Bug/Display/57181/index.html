  Mon May 03 15:59:19 2010 ambrus [...] math.bme.hu - Ticket created
Subject: INSTALL file misleading
The cpan module distro Module-Pluggable-3.9 has an INSTALL file which claims that the module uses Module::Build and so you have to install it using Build.PL. However, the distro does not actually come with a Build.PL, but a Makefile.PL (autogenerated by Module::Build::Compat, uses ExtUtils::MakeMaker). Thus, the instructions in INSTALL don't work. Please either fix the instructions in INSTALL or the distribution. Thanks. Other than this, I could install the module successfully by disregarding the advice of INSTALL. I am using a debian lenny x86_64 linux system with vanilla perl 5.12.0.
