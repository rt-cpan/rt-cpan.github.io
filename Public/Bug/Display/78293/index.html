Hailo is a program and library to generate new somewhat-random text based on other texts it has been fed. It is based on an old program called MegaHAL (http://en.wikipedia.org/wiki/MegaHAL). Hailo is mostly used to create chatbots (http://en.wikipedia.org/wiki/Chatterbot). It has also been used for things like http://www.tweetmix.me/. Hailo is dual-licensed under the GPL license (http://en.wikipedia.org/wiki/GNU_General_Public_License) and the Artistic license (http://en.wikipedia.org/wiki/Artistic_License), meaning you can use it in most open-source products. As for commercial closed-source use, we could license it to you at a fee for that purpose.
