Hi Miko, The release name for your Array::OneOf module is Array-OneOf, but in the META.yml file the dist name is given as: name: Array::OneOf Ie the module name, but this *should* be: name: Array-OneOf This can cause problems for tools / people automatically processing CPAN distributions, which is how I came across your dist. I generated a list of all such dists: https://gist.github.com/neilbowers/10060761 Would you mind regenerating the META.yml and doing a release with the correct metadata please? If you login into the PAUSE web site, then select "Change Permissions" on the left sidebar. Then click the "select" button next to "3.1 Make somebody else co-maintainer".
