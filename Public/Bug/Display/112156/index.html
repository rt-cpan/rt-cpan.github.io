Hi Marc, yes indeed this is a bug. Option attributes don't seem to be propagated into command classes unless your command classes inherit from your base class. As an immediate workaround I suggest to add 'extend qw(MyApp);' into you command class using cmd_term. I'll have a look into this issue in the meantime. Cheers Maro≈° Show quoted text
