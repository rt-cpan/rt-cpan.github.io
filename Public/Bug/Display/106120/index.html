Subject: Fails tests
Perl changed how it used isa around 5.9.3 days I believe, so this module failed tests. In addition, if "overload.pm" is not in %ISA on recent versions of Perl (well, more recent than this module), it gets loaded again which generates errors. The module works fine, but the tests need a patch. This is obviously a very important package - I wouldn't want anyone thinking the quality of Acme isn't up to snuff.
