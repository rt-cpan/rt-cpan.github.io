Subject: Do not package pm_to_blib!
Your distribution contains the pm_to_blib. This effectively makes it impossible to build and install the distribution. Please remove the pm_to_blib, then everything should work OK. I guess that you're not using "make dist" to create the tarball. Using "make dist" is highly recommended, because it avoids many of possible errors. Another useful make rule is "make disttest" to check if a distribution would extract, build and test OK.
