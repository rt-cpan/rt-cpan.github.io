In preparation for the release of perl-5.28.0 in May 2018, I have been testing a selected subset of CPAN distributions against monthly development releases in the perl-5.27.* series. Today I tested this distribution against perl-5.27.10 on FreeBSD-11.0 using 'cpanm' as the installer. Please see the attached excerpt from the build.log. Also attached is the relevant 'perl -V' output. I tried the patch provided in this ticket by Chad Granum in July 2016. It worked; all tests passed and I was able to install against perl-5.27.10. Other CPAN distributions have a dependency on Lexical-Var. We are unable to assess their readiness for perl-5.28 until we get a new CPAN release of this distribution which addresses the problems discussed. So application of the patch and a new CPAN release would be appreciated. Thank you very much. Jim Keenan
