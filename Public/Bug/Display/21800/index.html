There are only a few things preventing File::Copy::Recursive from being backwards compatible before Perl 5.6: * use warnings * lexical directory handles * our variables None of these are critical to the operation of the module itself. A fairly trivial patch will make this excellent module useful for Perls before 5.6. (See attachment.) I use this module for tests in CPAN::Reporter, and would really appreciate having improved backwards compatibility.
