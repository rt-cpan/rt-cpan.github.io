Hi, This is a low priority request to add to the existing WWW::Mechanize documentation found on: http://search.cpan.org/~petdance/WWW-Mechanize-1.11_02/lib/WWW/Mechanize.pm This is probably a very obvious statement, but for persons new to perl, screen scraping or automation tools, it may be of value to point out that when submitting forms, all fields should be included - regardless of whether they are visible or not, and filled in appropriately - Otherwise the form submital may not work, but will also not return an error - a potentially confusing situation. To summerize, please include, somewhere in the forms section of the WWW:Mechanize documentation a note stating that one must be sure to submit *all* fields of a form, including hidden values, in order to assure success in form submital. Best regards, Altrus
