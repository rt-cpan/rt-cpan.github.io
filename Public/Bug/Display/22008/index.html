This is a duplicate for 21898. It is not related to the platform, but to the version of libxml2 used (most likely 2.6.16), although none of the reports mentions it. This is not a bug nor it is related to the 1.61 version of LibXSLT. The regression test (which was added to the distribution only in 1.61) has to be modified in order to reflect differences between the output produced by different versions of libxml2. Will be fixed in 1.62 and soon in the SVN. In the meantime, ignore the failed test. -- Petr
