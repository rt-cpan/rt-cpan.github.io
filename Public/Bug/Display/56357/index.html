John-Paul, Thanks for your report. It looks like you have created your model using the 'database' shapes in dia. For historic reasons Parse-Dia-SQL supports the 'UML' shapes only. Please consult the t/data directory below for example models. http://cpansearch.perl.org/src/AFF/Parse-Dia-SQL-0.14/t/data/ Adding support for the 'database' shapes to PDS is probably a good long-term solution. Best regards, Andreas
