Subject: PATCH: improve compatibility with older Perls
This module uses "utf:is_utf8()", which was introduced in Perl 5.8.1. It is functionally equivalent to "Encode::is_utf8()", which was introduced in Perl 5.7.3. This patch switches use the syntax to use "Encode::is_utf8()", which allows the module to function on Perl 5.8.0, which is what I'm currently using. Mark
