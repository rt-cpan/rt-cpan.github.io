Subject: Support custom API urls.
Hi! Russian search engine Yandex (http://en.wikipedia.org/wiki/Yandex) maintains its own SafeBrowsing database with other API urls: http://api.yandex.ru/safebrowsing/doc/quickstart/tasks/integration_difftypes.xml Getting API key: http://safe.yandex.ru/keys/ (Yandex login required). Patch allow override api urls. Sample code: =========== use Net::Google::SafeBrowsing2; use Net::Google::SafeBrowsing2::Sqlite; my $storage = Net::Google::SafeBrowsing2::Sqlite->new(file => 'sb-yandex-v2.db'); my $gsb = Net::Google::SafeBrowsing2->new( key => 'my key', storage => $storage, version => 2.3, urls => { downloads => 'http://sba.yandex.net/downloads', gethash => 'http://sba.yandex.net/gethash', list => 'http://sba.yandex.net/list', newkey => 'https://sba.yandex.net/newkey', }, ); $gsb->update(); my $match = $gsb->lookup(url => 'http://www.gumblar.cn/'); $storage->close(); ===========
