Subject: fails to build on distros using perl less than 5.8.8
CSS::Minifier::XS fails to build for most *nix distros running Apache 1.x and mod_perl 1 simple patch backports for perl 5.6.1 an earlier, patch submitted to author September 2012. Attached again here for anyone facing similar issues with development on older systems. 4 lines that require perl 5.8.8 removed 3 lines in XS.xs substitute Newz for Newxz allows this module to be used on all current and most older *nix distros still running.
