No, you haven't, except uploading to fast :-) What happened? My automatic tester picked up your upload of Text-vCard-1.5, waited 4 hours foer it to arrive on CPAN and started to test it. It noticed the prerequisite, found it missing and went to install it. Text-vFile-asData-0.02 *had* been uploaded earlier, but had not yet arrived in the CPAN *index*. So the automatic prerequisite fetching installed 0.01 and that led to the failure you saw. I'm not sure if this is a bug in CPANPLUS 0.499, but I think after installing prerequisites it should have a look if the correct version arrived and suppress the error mail in the case of cpansmoke.
