It looks like I'm failing to handle PDF comments in several places. This quick workaround would probably work. It replaces comments with an equal number of spaces, so the change won't disrupt the internal byte indexing of the document. perl -i -pe's/^([ \t]*%[^\r\n]*)/ ' ' x length $1 /egms;' result.pdf (but I have not tested it against a real PDF document) In the longer term, I need to update my parser to allow comments.
