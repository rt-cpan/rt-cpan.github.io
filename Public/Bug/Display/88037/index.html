From b0ad6a1337bb2c020e264c7c3cf3f9d11afb9b90 Mon Sep 17 00:00:00 2001 From: bulk88 <bulk88@hotmail.com> Date: Thu, 22 Aug 2013 15:51:59 -0400 Subject: [PATCH] regen run of Makefile.PL will use same -MCross arg as initial .PL run This is for the WinCE build process. If -MCross was used on the 1st run of Makefile.PL, from buildext.pl for example, any regen runs done by makefile, must call Makefile.PL the same way that was Makefile.PL was initially called.
