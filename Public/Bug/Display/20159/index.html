Wheel::ReadLine won't be very helpful on Windows. ActiveState Perl can only multiplex sockets, and your console isn't one. Nevertheless, I've corrected the B38400 error... and added an explicit die() if you're using Wheel::ReadLine in MSWin32. Thanks for the bug report.
