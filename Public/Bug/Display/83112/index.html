Feature request (patch attached) I wanted to be able to see the unicode flag for strings. (utf8::is_utf8) I also wanted to be able to render control characters and unicode as pure ascii escapes, for debugging. I added new options show_unicode escape_non_ascii I also re-implemented _escape_chars to be more efficient on average (I think, not tested) in addition to supporting these new features.
