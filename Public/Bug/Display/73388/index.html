Subject: Listener IPV6_V6ONLY control
Hard to do IPv4+IPv6-clean listening sockets in a portable manner. Need control of the IPV6_V6ONLY sockopt. Consider $listener->listen( addr => { family => "inet6", port => 1234, ip => "::", v6only => 0 }, ... ); -- Paul Evans
  Mon Mar 12 15:00:22 2012 leonerd-cpan [...] leonerd.org.uk - Correspondence added
