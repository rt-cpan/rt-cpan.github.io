Subject: better memory handling
Hi. CSS::Minifier::XS 0.03 may sometimes cause segfaults under my environment (Win2K+ActivePerl 5.8.7+gcc3.4.2; actually "always" for the condition written in t/99-benchmark.t). Using Perl's alternative New(z)/ Safefree macros (instead of built-in malloc/free) seems to remove this problem constantly. A patch attached. Thanks. Kenichi Ishigaki
