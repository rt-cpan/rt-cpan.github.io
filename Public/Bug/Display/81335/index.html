Subject: [PATCH] Fix with Test::Builder::Tester 0.32
Test::Script requires Test::Builder::Tester, the version which came with 5.8.8, but tests do not pass with that version. The test_err('/.../') regex syntax was broken for a period ending with Test::More 0.66. Since I don't think Test::Script needs to work before qr// is available, it can just use that.
