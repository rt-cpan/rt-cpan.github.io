Subject: Worksheet State not included in data structure
The data structure the code returns does not include the State of each worksheet. What I have done is attach a proposed change that will provide this. The change is around line 132 of XLSX.pm (attached) and reads: elsif ($k eq 'state') { $sheet -> {State} = $v; } This is then accessible in code using something like: my $excel = Spreadsheet::XLSX -> new ($excelFileName, $converter); say $excel->{Worksheet}[0]{State}; I need this feature as I am working on a spreadsheet parser that is only interested in visible worksheets.
