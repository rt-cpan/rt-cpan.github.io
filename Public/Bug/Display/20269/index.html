> The code hangs at > my @list = $ftp->dir;
I doubt that running FC5 is the issue, I have Net::FTP running on several FC5 boxes. Is it maybe the connect is timing out? How long do you wait? the default timeout is going to be 2 minutes. You can lower that by passing Timeout => 5 to the constructor. Another thing to try would be to try not using Passive by passing Passive => 0 to the constructor. Graham.
