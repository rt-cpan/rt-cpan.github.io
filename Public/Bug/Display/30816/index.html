Subject: Don't escape HTML meta-characters
Text::Trac SVN trunk (rev. 370) don't escape HTML meta-characters. So I wrote a patch to fix this. In the patch, Mainly I added escape() to InlineNode and BlockNode. And modified parse() on InlineNode to call escape().
