Subject: DANE TLSA support
This patch adds support for DANE TLSA records conforming to the current draft, draft-ietf- dane-protocol-21. It adds the TLSA record with the IANA assigned qtype of 52 and a module Net::DNS::RR:TLSA. Adding this now may be a bit early and I can imagine limiting Net::DNS only to records which have an actual standard associated with them. Support was added to our local version for a customer who wants to test with DANE, thought it would be nice to share it.
