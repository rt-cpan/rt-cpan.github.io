Subject: Support Devel::StackTrace max_arg_length feature
In Devel::StackTrace #33519, I requested a new max_arg_length option to limit the size of arguments displayed. You added it in 1.18. Exception::Class needs a way to pass this option to the Devel::StackTrace objects that it creates. Attached is a patch that adds the support.
