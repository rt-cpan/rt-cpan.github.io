Hello, when using inline::java 0.51 with JDK 1.4 you got an Unsupported.major.minor.version 49.0 exception. the problem ist that the jar files "InlineJavaUser.jar" & "InlineJavaServer.jar" are precompiled with jdk 1.5 or later. a workarround for this issue is to repack those jar files using the desired JDK version (happiliy, the sources are in the inline/java folder). a way to solve this issue is to pack those jars AFTER donwloading to the client machine with the given JDK during installation, so the enduser always have the jar files in appropriate java version. best regards, daniel mÃ¼ller Show quoted text
