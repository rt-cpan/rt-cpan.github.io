Both Shorl and TinyURL have changed their access to URLs. Shorl now use a intermmediate page and TinyURL use 'forwarding' as a subdomain instead of 'redirecting'. The file here patches Shorl.pm and tinyurl.t.# In addition please note that 'http://perl.dellah.org/WWW-Shorten-1.5.2.tar.gz' is now returning a 404 HTTP error, and as such notlong.t now fails. On further investigation, the page that notlong.com sends back contains the phrase 'Check the long URL: 404 Not Found', so you might want to check for this in future versions and return undef.
