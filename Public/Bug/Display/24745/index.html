HTML::WikiConverter (0.62) Via website http://diberri.dyndns.org/wikipedia/html2wiki/index.html Converting an html document to MediaWiki resulted in huge letters in my Wiki page. Doing some research, I noticed that <span> tags in the html source were converted to <font> tags in MediaWiki markup. The MediaWiki documentation states that the font tag has been deprecated in favor of span, so I reckon this translation should be removed. Also, <font> tags like <font size="8.0pt"> are not proper html. Html supports size form 1 to 7. ================================ Input: <html> <head> <meta http-equiv=Content-Type content="text/html; charset=windows-1252"> <meta name=Generator content="Microsoft Word 11 (filtered)"> <title>Quality Manual</title> </head> <body lang=EN-US link=blue vlink=purple style='text-justify- trim:punctuation'> <div class=Section1> <p class=MsoTitle><span style='font-size:40.0pt;font-family:"Arial Narrow"'>The Test Header</span></p> </div> </body> </html> ================================ Result: <div class="Section1"> <font face="&quot;Arial Narrow&quot;"><font size="40.0pt">The Test Header</font></font> </div>
