This was the flag that maybe Pod::WordML wasn't declaring its dependency on Pod::Escapes. I just checked and noticed that Pod::WordML didn't declare the dependency in 0.13, but the 0.14 release did declare the dependency. So if you install that, it should pull in Pod::Escapes and everything should be fine. Closing here as not an issue with Pod::Escapes.
