The documentation for the module claims to give examples of using fgrep() in void, scalar, and list context. However, the first example is not void context. It is a boolean context (which is a subset of scalar). That example is given as: if ( fgrep { /$user/ } "/etc/passwd" ) { do_something(); } In this example, the return value of fgrep() is being used as the condition to the if(). That is a boolean context.
