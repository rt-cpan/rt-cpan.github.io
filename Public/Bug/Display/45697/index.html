Subject: Support for old C (Sun Studio / MS VC) compilers
When using older compilers (Sun Studio before SunOS 5.10 or MSVC7/8) build fails if the code contains declarations outside of file scope or the start of compound statement. (allowed in C99) The attached patch against Libdnet.xs allows the code to be compiled with these compilers.
