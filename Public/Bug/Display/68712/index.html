First of all... Your module has been a lifesaver. A Million thanks! However, for an application I am developing, I needed to add support for the "QUOT" command to AutoReconnect.pm. This command allows your module to use any FTP command not directly supported by NET::FTP http://perldoc.perl.org/Net/FTP.html#Methods-for-the-adventurous Added (Line 524 of AutoReconnect.pm): sub quot { my $self = shift; $self->{ftp}->quot(@_); } Usage Examples: $ftpcon->quot('SITE', sprintf('CHMOD %04o %s', (lstat $curlocaldir)[2] & 07777, $curlocaldir)); $ftpcon->quot('MFMT', $iso8601_timestamp, $filename1); $ftpcon->quot('MDTM', $iso8601_timestamp, $filename1); The above examples issue SITE <http://www.nsftools.com/tips/RawFTP.htm#SITE> , MFMT, and (non-standard syntax) MDTM commands to a remote server, without specifically adding support for each these commands individually to Net::FTP::AutoReconnect. Please consider adding this extremely useful "catch-all" feature to your code. Chris Schuler
