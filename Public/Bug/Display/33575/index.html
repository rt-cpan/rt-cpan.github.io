Hi, The attached patch demonstrates some issues with blockquotes, HR, and Citations. Blockquotes: Multiparagraph block quotes capture only one paragraph. So for this: Ask not what your country can do for you. Ask what you can do for your country. --John F. Kennedy Only the second paragraph appears in a blockquote. They should both appear in the same blockquote. Note that Trac doesn't get this quite right, either. See http://trac.edgewall.org/ticket/6887. HR: According to the Trac documentation, "----" should create an HR, not "====". See http://trac.edgewall.org/wiki/WikiFormatting#Miscellaneous. Citations: Looks like Text::Trac doesn't support email-type citations at all. See http://trac.edgewall.org/wiki/WikiFormatting#DiscussionCitations . Thanks, David
