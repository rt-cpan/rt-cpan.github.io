text/plain 594b
Hi! C::C::HTML::FormFu::BUILD misses individual options from the application configfiles. It fetches the controllers config with: $self_config = $self->config->{'Controller::HTML::FormFu'} that is wrong. This is just the __PACKAGE__->config('Controller::HTML::FormFu'=>stuff). Instead it should use: $self_config = $self->{'Controller::HTML::FormFu'} This returns the merged options of __PACKAGE__->config() and Catalysts app. configfile, so now u can also set up controllers individual: MyApp.yml Controller::MyFormStuff: Controller::HTML::FormFu: ... options ... greetz
