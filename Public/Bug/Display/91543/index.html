Hi, I've compiled my own Perl 5.18.1 on AIX 7.1 TL2 SP3 with IBM XL C/C++ v12.1.0.5 I've installed a few additional modules and one of them failed a test. I've reported the error encountered: https://groups.google.com/forum/#!topic/mojolicious/T34XZX72LaA After a few tests, we've been able to target the issue in the getsockopt() subroutine of IO::Socket. It seems that this subroutine does not meet the condition to unpack on AIX while it should do it. I've opened a ticket against IO::Socket: https://rt.cpan.org/Public/Bug/Display.html?id=91183 However, it seems that IO::Socket is now considered as part of Perl5 core and therefore this is why I'm opening this ticket today.
