Subject: support for authkey parameter
I'd like the Picasa module to support fetching images from a hidden album, by using a provided authkey parameter. I've attached a patch that adds this functionality. See also http://code.google.com/apis/picasaweb/faq.html#unlisted_albums It'd be great if this was included in a future release. Thanks for module!
Subject: Photo.pm.authkey.patch
