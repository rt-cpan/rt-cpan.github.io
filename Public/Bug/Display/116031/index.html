Hello. First of all, thank you very much and congratulations for your module BetaCode. I am writing you because I think there are a couple punctuation signs missing. One is the apostrophe ( betacode: ' unicode: ’ ). It is the same betacode symbol as for "breve", but it translates differently when it doesn't follow a vowel. The other is the numeral ( betacode: # unicode: ʹ ) You can find the apostrophe in Wikipedia's "Beta Code" article. The numeral is not there, but you can see it in many CCAT texts, like for example this one: http://goo.gl/aE4d60 in betacode, http://goo.gl/gFSNKa rendered to Unicode (scroll down to verse 32). I am attaching a modified version of the module, in case you want to adopt the changes. Thanks. Best regards, Juan.
