I think this problem is caused by timezone. The failing test is: comparing that "2014-01-02" is less than "2014-01-02T02:10:10Z" but the former uses local time zone (timelocal() function) while the latter uses timegm(), so the test might succeed or fail depending on the local timezone. I will have to fix the test case.
