The error message seems to indicate that the bool "type" is not defined and that's usually defined in stdbool.h. The first question then is whether libcdio compiles on OSX Lion. If so you there should be a cdio/cdio_config.h include file and that will have a line that indicates whether libcdio thinks you have the stdbool.h header. On my system at line 199 I have: /* Define to 1 if you have the <stdbool.h> header file. */ #define HAVE_STDBOOL_H 1 When this is not defined libcdio uses its own workaround defines. So my guess is that you have HAVE_STDBOOL 1 as above, but your stdbool.h doesn't define bool.
