I recently "upgraded" from v2.12 to v2.16 and am seeing horrible performance there as well. I use Storable to save large data caches (50-100 MB) and v2.16 seems to write only about 20 kB per second! This is on WinXP Pro with SP2. Downgrading to v2.14 fixed the performance issue.
