The question is why the test wasn't skipped, given the logic in the test suite: } elsif ($n == 4 && $Pod::Simple::VERSION < 3.06) { print "ok $n # skip Pod::Simple S<> parsing bug\n"; Could you double-check what version of Pod::Simple you have installed?
