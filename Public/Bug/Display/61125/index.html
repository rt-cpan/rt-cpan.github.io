"Although this is level of nesting is" -> "although this nesting is" "was expanded" -> "were expanded" (subjunctive needed here) "labeled labeled" -> "labeled" "XML:LibXML" -> "XML::LibXML" The function "load_html" is incorrectly headed "load_xml". After this function, the indenting goes wrong: all the rest of the parsing functions are indented by one extra level. The indenting should return to the top level at the paragraph starting "The functions described above are implemented to parse well formed documents." 2. XML::LibXML::Element parse_xml_chunk -> parse_balanced_chunk (use the current name, not the old one) 3. XML::LibXML "considered a bad practice" -> "considered bad practice"
