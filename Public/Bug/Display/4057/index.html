Hi, I tried to use backticks with Net::Server::PreFork. The script always crashs. So I tested it with fork, exec and waitpid but it crashed too. The interesting thing was that it sometimes failed and sometimes succeed. The successrate was between 40 and 80 percent. I was ready to give up but then I thought take it small and simple and tested the superclasses PreForkSimple and Fork. Surprisingly PreForkSimple failed too but Fork didn't fail. I checked the code in PreForkSimple and it looks like a problem with the communication between the server and the preforked childs. I attached a sample script which works with Fork and crashs with PreFork(Simple). I hope this helps a little bit.
