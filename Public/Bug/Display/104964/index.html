Subject: Reproducible build
While working on Debian's “reproducible builds” effort [1], we have noticed that this module doesn't build reproducibly because Makefile.PL is using hash keys in a random order. Please consider the attached patch, which fixes this. [1]: https://wiki.debian.org/ReproducibleBuilds This is also https://bugs.debian.org/787754 Thanks for your work on free software, -- Niko Tyni ntyni@debian.org
