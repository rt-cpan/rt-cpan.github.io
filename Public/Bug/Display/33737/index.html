Hello, This code: Show quoted text
> /* Strip final newline so line number appears for warn/die */ > if (err[strlen(err)] == 10) > err[strlen(err)] = '\0';
is simply wrong. Note that, whenever the "err" string is null-terminated, err[strlen(err)] is always '\0' (e.g. if strlen(err) == 3, then the three characters are indexed by 0, 1, and 2; and err[3] is final '\0'). The attached patch fixes this case by using sv_setpvn instead; and also another case, by using snprintf(3). -- Alexey Tourbin ALT Linux Team
