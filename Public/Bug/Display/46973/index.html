Subject: all_pod_files_ok() doesn't recognize .bat files created by pl2bat as perl
On Win32, the _is_perl function does not match .bat files which are perl programs converted to .bat with pl2bat. So scripts are not checked on Win32. _is_perl only checks that the first line of a file with an unknown extension matches /^#!.*perl/. It should also check for '--*-Perl-*--'.
