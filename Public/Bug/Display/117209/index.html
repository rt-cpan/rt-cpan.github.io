As I understand, the license you mention is the original Artistic 1 license <http://dev.perl.org/licenses/artistic.html>. This license is not acceptable for some parties, e.g. Fedora Linux distribution, because it's too vague <http://www.gnu.org/licenses/license-list.html#ArtisticLicense>. The Perl has dual-licensing with GNU GPL <http://dev.perl.org/licenses/> . If you want to distribute it in "same term as perl" the "GNU General Public License" should be also mention.
