Hi Michael, I am not using EMACS, so I will not implement it on my own, but I am happy to help everybody who is willing to get it working. As far as I see there is an Language Server for EMACS. As soon as the EMACS Language Server Client is running, it should be possible to use the Perl::LanguageServer by issuing the following command line perl -MPerl::LanguageServer -e Perl::LanguageServer::run for the upcoming debug protocol adapter the command line should look like this perl -MPerl::LanguageServer -e Perl::LanguageServer::run -- --port 13603 but in this case you would also need a Debug Adapter Client for EMACS (and debug extention for Perl::LanguageServer is not ready yet) Regards Gerald Show quoted text
