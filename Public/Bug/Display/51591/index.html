WKS 11 a well known service description [RFC1035] Is said to be deprecated. GPOS 27 Geographical Position [RFC1712] Is an experimental record with a rather confusing implementation status (and a specification bug that may cause interoperability problems, see http://www.rfc- editor.org/errata_search.php?rfc=1712. The LOC record supplies all functionality that GPOS intended to provide. KX 36 Key Exchanger [RFC2230] Implemented on the trunk DHCID 49 DHCID [RFC4701] Implemented on the trunk HIP 55 Host Identity Protocol [RFC5205] Implemented on the trunk I expect a release shortly. FWIW, from the currently assignments by IANA the following are not implemented (seee http://www.iana.org/assignments/dns-parameters) 'MD' => 3, # RFC 1035, Section 3.3.4 (obsolete) NOT IMPLEMENTED 'MF' => 4, # RFC 1035, Section 3.3.5 (obsolete) NOT IMPLEMENTED 'WKS' => 11, # RFC 1035, Section 3.4.2 (deprecated) NOT IMPLEMENTED 'NSAP_PTR' => 23, # RFC 1348 (obsolete by RFC 1637) NOT IMPLEMENTED 'GPOS' => 27, # RFC 1712 (obsolete ?) NOT IMPLEMENTED 'ATMA' => 34, # non-standard NOT IMPLEMENTED 'A6' => 38, # RFC3226, RFC2874. See RFC 3363 made A6 exp. NOT IMPLEMENTED 'SINK' => 40, # non-standard NOT IMPLEMENTED 'NINFO' => 56, # non-standard NOT IMPLEMENTED 'RKEY' => 57, # non-standard NOT IMPLEMENTED 'TA' => 32768, # non-standard NOT IMPLEMENTED For those marked non-standard the specification may be hard to obtain.
