Subject: Indentation pains for no apperant reason

Thsi seems related to my earlier reported bug with the UTF8 encoding as the only XML I have found to have issues with indentation is XML in which the extended characters are used. To test this create two XML messages one with UTF8 encoding but no special characters and one with UTF8 encoding and some special characters, when parsing the file witht the extended characters will not indent correctly.
