Subject: Move to webkitgtk4
Fedora 27 Linux distribution removed webkitgtk3 RPM package (WebKit1 with Gtk3) in favor of webkitgtk4 RPM package (WebKit2 with Gtk3) due to security reasons. Please consider switching this Gtk3-WebKit Perl binding too. See the attached patch.
