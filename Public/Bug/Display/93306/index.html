Subject: [PATCH] cas gets support
The attached patch adds support for parsing cas results from "gets" operations. It is compatible with the Cache::Memcached updates posted to RT# 93304. It was necessary to modify GetParser.pm (and this GetParser.xs) in order to support the cas operations that have been present in memcached since 2007-12-06 v1.2.4 (current version is 1.4.17). Please note, I also noticed that utf8 key support is broken. I have not fixed that .
