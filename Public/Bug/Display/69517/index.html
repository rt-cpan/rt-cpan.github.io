Subject: New version of Tkpp (Version 1.2)
Dear, I am work on Windows, Linux... but for the moment, I am working on Vista system with ActivePerl 5.12. I regularly use the pp programme. In recent days, I decided to use tkpp and I realized it does not handle all the pp options. Having used to work under Perl/Tk, I took the initiative to develop a new tkpp application. I attach to this ticket some screenshots of the new tkpp application. It supports all the pp options. The application is divided into several tabs in order to have a reasonable window size. The application allows you to save in a file the command line used to generated package, to save a configuration file containing all options you selected in the interface to reload it later. For users working in Windows, tkpp now creates the package through another process, so that, the interface does not freeze. For other OS (Linux, MacOS ...), it is not the case, a perl system command is used. I could have used a thread, but I did not see fit to do so. So, if you want to test it for inclusion in a future release of PAR:: Packer, let me know and I'll put the code in the ticket. While waiting for your response, I enclose the POD and screenshots. Best regards, Djibril (DJIBEL)
