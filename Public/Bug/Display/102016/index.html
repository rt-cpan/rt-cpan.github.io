Hi, Please see the attached: i) test file ii) sample output iii) perl version details. DBI version: 1.628 DBD::mysql version: 4.029 mysql server: 5.6.14 This only happens when using the debugger. The program runs fine otherwise. I assume this is a DBD::mysql issue as the transactions are handled by it. There are some posts mentioning this issue dating back to several years, in which one person suggests calling the method in list context, and it indeed removes the error: my($rv) = $dbh->commit; Thanks, Burak
