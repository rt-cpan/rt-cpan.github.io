Subject: C plugin: Integers incorrectly detected, 0/4 ignored, typedef ignored
The attached script shows Syntax::Highlight::Engine::Kate::C failing to detect the integers 0 and 4. Other positive integers are detected as 'float'. Negative integers are correctly detected as 'decval'. 'typedef' is also ignored completely. ANSI_C89 mode also shows the same problem.
