Hi - Great package! I'm in the process of porting my Ruby debugger to Perl and would like to add Enbugger support for that package. Attached should be a patch to add support. The code is at https://github.com/rocky/Perl-Devel-Trepan. It is far from finished, but even at this early stage it has some features that other debuggers don't have such as syntax highlighted via Syntax::Highlight::Perl::Improved, GNU Readline completion of debugger commands, extensive on-line help, support for inspecting and changing a return value of a function and is use extensible, modular, and testable. The package use Array::Columnize which I've just uploaded to CPAN. It too can be found on github at https://github.com/rocky/Perl-Array- Columnize.
