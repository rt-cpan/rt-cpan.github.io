Subject: parsed description still contains POD commands (=over,=item, etc)
Using cpan2rpm v2.028, if you issue cpan2rpm http://search.cpan.org/CPAN/authors/id/G/GA/GAAS/libwww-perl-5.805.tar.gz POD commands (like =over and =item) are still visible on the %description section of the rpm specfile. Also, "None." would appear if you try it on the latest CGI tar ball. A proposed patch (based on Pod::Parser) is attached. HTH Sherwin Daganato
