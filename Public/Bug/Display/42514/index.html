Subject: [PATCH] Add tests which actually test PDF::FromHTML can render various HTML files
This patch (against 0.26) adds several tests against PDF::FromHTML, to check that it can successfully render a number of simple HTML files. The maintainers of this module may be interested to note that one warning and one failure emerge from these simple tests. I will open a separate bug report for the failure.
