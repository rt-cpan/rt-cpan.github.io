  Tue Oct 22 15:34:26 2002 gemerson [...] vizdom.com - Correspondence added
It's not so much the Perl version as the DBI version. DBD::JDBC uses a constant which used to be defined in the DBI module but was removed after DBI 1.14. A workaround is to replace uses of SQL_BIGINT with -5 in JDBC.pm.
