Subject: New Feature: Allow XML::RSS objects in sources list
The attached patch allows you to pass in XML::RSS objects in the sources list. The reason for this is that I want to collect the RSS feeds myself, using mirror() to be more bandwith friendly, instead of the default get(). I've not tested this patch (just about to...), but you may find it useful. Thanks, -Dom
