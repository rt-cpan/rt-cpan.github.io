There are two problems with $DEBUG. 1) If it is set to a false value debugging will be turned on. This is contrary to the docs and common sense. 2) $DEBUG should be set in a BEGIN block in the SYNOPSIS so it is set before the use Class::Autouse happens. Patch attached.
