CGI-FormBuilder-3.0401 Using a select field with the "other" option, if the field has a value which is not in the options list, then the "other" text input field is correctly enabled and filled in, but the select field is not set to "Other:". This means that a subsequent submit will reset the field to whatever is the default for the select.
