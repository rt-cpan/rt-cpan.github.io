The attached patch adds support for Module::Build. I tested it out on one of my projects and it worked well. It's version-finding code may not be 100% correct for all possible cases, but I think it's decent. It'd be better to run the Build.PL, then use the Module::Build API to get the version (via the dist_version method), but I'm not sure you wanted to do that.
