Subject: Ability to submit forms that don't have a submit button or image.
I have been sucessfuly using your WWW::Mechanize module to load photos to Ofoto.com. It is very cool. One aspect of the Ofoto.com site is that a lot of their forms do not have a submit button or image. Instead they have an image in an href with javascript to submit the form. I can't use the submit() or click() methods because of this. Instead I have to: o call make_request() on the form o call _push_page_stack on the WWW::Mechanize agent o set the uri for the agent o set the req for the agent o call _do_request() It would be nice if the submit() method would do the above if no submit-able input was found.
