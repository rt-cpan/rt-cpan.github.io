The included patch uses the much more portable Sys::Hostname instead of `hostname`. It also includes a new option to docfiles: format:pdf (or tiff, or...) which sends a FORM request to the server, which means the server will treat following documents as that format, rather then autodetecting the format. Incudes documentation patch, but no unit test
