Subject: PathTools v3.13 breaks Module::Find v3.04
PathTools v3.13 (and later) changed the behavior of abs2rel, which breaks Module::Find. Here's the entry from the Changes file for PathTools: - abs2rel() used to return the empty string when its two arguments were identical, which made no sense. Now it returns curdir(). [Spotted by David Golden] The diagnostic is that the tests fail.
