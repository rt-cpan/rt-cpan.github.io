Hi, Please find attached a patch that converts é ("e" with small line above, falling from right to left) in author's name from Latin-1 to utf-8 encoding. It is used in the Debian package of File-Read, as Debian aims at 100% utf-8 content. Please apply it to the next release of File-Read if you find it useful. Oh, and in case that I got the character wrong, please accept my apologies and tell me what it should be. Thanks, dam
