Subject: tests fail on ActiveState 5.8.8 (822) -- upgrading EU::Install fixes it
I tried installing Bundle::CPAN on a stock, out-of-the-box ActiveState Perl 5.8.8 (build 822) and ExtUtils::MakeMaker kept failing tests in basic.t. It looked like it might be related to ExtUtils::Install, so I upgraded that to 1.44 and then EU::MM passed all tests. You probably need to bundle a higher version of EU::Install in the next release.
