From 96209b5ba3df009a495da31bee8e2540c3365ec5 Mon Sep 17 00:00:00 2001 From: Niko Tyni <ntyni@debian.org> Date: Sat, 17 Sep 2016 21:16:16 +0300 Subject: [PATCH] Fix a race condition in the test suite waiting for daemon log output There's a window where the daemon has created a log file but not yet written anything into it.
