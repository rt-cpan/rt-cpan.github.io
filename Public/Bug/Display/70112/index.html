Subject: GetBitmap returns partial bitmap
Method GetBitmap does not return the entire image - it seems only the bitmap header (along with the file header generated by the module) is returned. Further investigation shows that the field miHeader.biSizeImage is zero in the data returned my the system call GetClipboardData. The attached patch replaces the field with GlobalSize (handle returned by system call) if it is zero.
