Subject: allow for disabling version output when author testing
I would find this plugin much more useful if it did not run during development (i.e. during AUTHOR_TESTING) -- I don't care to see a long list of versions when iterating through tests, as it causes other output to scroll past the edge of the screen. This patch -- https://github.com/dolmen/p5-Dist-Zilla-Plugin-ReportVersions-Tiny/pull/6 adds a configuration option which turns the plugin off during AUTHOR_TESTING. It defaults to off, so current users are unaffected. The setting must be explicitly set for there to be any change in operation. Please merge this patch, as I would like to add it to my author plugin bundle! thanks.
