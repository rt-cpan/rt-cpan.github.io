The issue lies in MakeMaker's idea of how long a command can be in Windows. Currently the values are set at 2500 for Win9X and 31K for everything else. These values are based on experimentation with Win98 and Windows2000 respectively. Since XP falls into "everything else" XP can't handle a 31K long command line. Since I don't know where to find this information, it has to be derived experimentally. I don't have an XP machine to experiment with. Could you download and install ExtUtils::MakeMaker 6.15 and play with the value in ExtUtils::MM_Win32::max_exec_len() until you get the highest number which will work? -- Michael G Schwern schwern@pobox.com http://www.pobox.com/~schwern/ I need a SHOWER a BURGER and some ROBOTS, STAT! -- http://www.angryflower.com/allrigh.gif
