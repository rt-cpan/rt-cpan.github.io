Dear Ian, I am trying to setup a SNMP request page for some users in our intranet. It's nothing special, only a cgi based page, you have to type in hostname und get the results back. My problem is that, if you enter a host, on which isn't running snmpd or Linux, you get a timeout but no error code I could handle. Is there any hidden one I haven't found or how can I handle this? It's running under CentOS 5 with perl 5.8.8, Thanks in advance, Sebastian
