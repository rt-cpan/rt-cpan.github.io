Hi Klaus and Stu, Sorry for a delay in replying to this. On Windows, there is a problem with building Gimp-Perl because to do so requires the GIMP headers, and the standard Windows distro for GIMP does not include those. My plan for dealing with this now will be to make Gimp-Perl interface with GIMP using the NativeCall module rather than the XS that it currently uses. If you would like to beat me to it and make a patch and post it on here, that would be very welcome! Ed
