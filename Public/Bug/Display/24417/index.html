Though the documentation mentions it briefly, it is currently (as of Graph-0.80) impossible to use the next_successor parameter when creating a new Graph::Traversal::* object (it dies with "unknown attribute"). The attached patch enables, documents, and tests this feature. Thanks! - Ted Carnahan
