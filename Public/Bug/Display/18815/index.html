text/plain 319b
The paths Crypt-OpenPGP uses to find keys aren't windows friendly. Attached are patches to address this issue. $ENV{HOME} doesn't exist on windows, so I've changed it to use File::HomeDir And I've also changed it to use File::Spec This is tested on perl 5.8.8, build with mingw gcc 3.4.5, on windows XP (not cygwin)
