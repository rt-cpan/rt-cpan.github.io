File::Find::Rule version 0.33. I'm not sure if this is a bug, but I was looking at the code, and it seemed to me that the _flatten function wasn't quite right. sub _flatten { my @flat; while (@_) { my $item = shift; ref $item eq 'ARRAY' ? push @_, @{ $item } : push @flat, $item; } return @flat; } Shouldn't that 'push @_, @{ $item }' be 'unshift @_, @{ $item }'. Otherwise you are reordering as well as flattening.
