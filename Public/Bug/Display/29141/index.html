I have a method that takes mixed paramaters (positional, followed by named). I first shift off the positional parameters, then run validate on the remainder of @_. But I'd like to also use validate_pos on the positional parameters. Ideally, I'd like to do: my ($data, $count) = (shift, shift); validate_pos( [$data, $count], {...}, {...} ); but I currently have to settle for this kludge because of the requirement of a named array: my ($data, $count) = (shift, shift); my @pos = ( $data, $count ); validate_pos( @pos, {...}, {...} );
