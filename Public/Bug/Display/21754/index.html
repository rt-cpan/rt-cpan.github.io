I was able to rule out the other modules you used, and in fact I reduced the test case to a very simple, manageable few lines: #!/usr/bin/perl use Tk; use POE; use warnings; use strict; sub _start { $_[KERNEL]->yield('redraw_graph'); } sub redraw_graph { $_[KERNEL]->delay_add( redraw_graph => 1 ); } POE::Session->create( inline_states => { _start => \&_start, redraw_graph => \&redraw_graph, } ); POE::Kernel->run; With this, I was able to duplicate and isolate the problem, then test that the fix worked.
