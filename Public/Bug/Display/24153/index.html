Subject: Test::TestCoverage breaks Devel::Cover
Devel::Cover report generation breaks when Test::TestCoverage is in use. This is not really surprising if you are familiar with the inner workings of both modules but not really obvious if you just want to use them. I guess this should be mentioned somewhere prominently in the docs if it isn't fixable somehow. A quick workaround is to disable calls to test_cover() if Devel::Cover is loaded (e.g. with "test_coverage('Foo') unless $Devel::Cover::VERSION;").
