Config::General 2.44 Perl v5.10.0 Linux 2.6.29.6 Config::General: EndBlock "</VirtualHost>" has no StartBlock statement (level: 1, chunk 5117)! When parsing an Apache configuration file, a <Directory> directive within a <VirtualHost ...> directive causes the above error IF the <Directory> directive includes a path with a trailing slash.
