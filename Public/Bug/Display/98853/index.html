Hi, Yes, the test reports the missing dat file and in your case 'brew reinstall geoipupdate’ should install the file. I think your geoid-c-api was not installed correctly. Homebrew has used GeoIPCountry.dat instead of GeoIP.dat by mistake. This bug is already fixed. You should run ‘brew reinstall geoipupdate’ to install the fixed version. I think that was the reason for your failure. Show quoted text
