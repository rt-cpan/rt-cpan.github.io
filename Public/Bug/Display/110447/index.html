This seems to be a problem of how you install CPAN modules. Using the "cpan" tool as an installer,t he documentation will be installed in a way that it is accessible for "perldoc". If you just download and unpack the distribution, this will not work. But you could still either: a) use "perldoc <full_path_to_the_pm-file>" or b) browse the documentation online at cpan.org Best regards, Darko
