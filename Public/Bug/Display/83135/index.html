RFC 6376: the "g=" tag has been deprecated in this version of the DKIM specification (and thus MUST now be ignored), signers are advised not to include the "g=" tag in key records... Request for Comments: 6376 Obsoletes: 4871, 5672 DomainKeys Identified Mail (DKIM) Signatures C.2.
