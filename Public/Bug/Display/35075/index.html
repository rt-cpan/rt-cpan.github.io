OK, this bug should be closed. This is mea culpa. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor\AutoRun on my computer changed to a folder of my preference. This caused all subshells to go to that folder. For instance when I did "look XML::Simple" in cpan, I would not be in the folder it said I was, but in the Command Processor\Autorun folder instead. This was the clue and ultimately the explaination is that "ERROR: Cannot Copy 'lib/XML/Simple/FAQ.pod' to 'blib\lib\XML\Simple\FAQ.pod': No such file or directory" was because it couldn't find the source lib/XML/Simple/FAQ.pod by relative path because my shell had changed the current folder from where the Install scripts expected it to be. Please closed this bug and accept my apologies.
