Subject: YAML reference tags winding up in JSON data
Sometimes YAML reference tags will wind up in JSON data. For instance, the following code produces JSON that contains syntax errors: perl -MJSON::Syck -MLWP::UserAgent -e 'my $ua = LWP::UserAgent->new; my $r = $ua->get("http://www.google.com/"); print JSON::Syck::Dump($r);' > example.txt The resulting code is attached. note the "!perl/" sections thrown into the middle of hash declarations. Thanks, Tyler
