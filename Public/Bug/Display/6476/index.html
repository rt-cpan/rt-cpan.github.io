Subject: Add getline method
It would be nice to add a "getline" alias for the "readline" method to achieve API compatibility with IO::Handle. That way I don't have to write my $nextline = $LOGFILE->can("readline") ? sub { $LOGFILE->readline } : sub { $LOGFILE->getline }; to be able to read forward or backward in the same piece of code. See the attached patch. Regards, Slaven
