Subject: PATCH: Document CGI->new, not "new CGI"

text/plain 316b
It is considered a best practice to create objects in this syntax: CGI->new(); rather than this: new CGI(); Because many people use the CGI docs for example code, the attached patch updates all the places in the CGI.pm code where this done to use the recommended syntax. Thanks for maintaining CGI.pm, Lincoln.
