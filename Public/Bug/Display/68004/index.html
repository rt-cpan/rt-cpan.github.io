> prereqs '3'
You have prereqs set to 3, which means 'build only'; this is the setting you want if you're testing/smoking modules. If you want to install them on disk you want 1 (yes) or 2 (ask). Typing 's conf reconfigure' can help you with this. This also means that the 'make' you are seeing is not for URI::FromHash, but for the module that depended on URI::FromHash; the build of U::FH succeeds, a test report is sent for it and then 'make' is ran for MooseX-Types-URI-0.02. You can verify this by running 't URI::FromHash'. Cheers, Show quoted text
