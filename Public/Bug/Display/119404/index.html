Volker, I probably need to improve the wording for those options. The path-to-rsync option is for the perl module, and overrides use of $PATH. The rsync-path option belongs to the rsync command, and affects how the rsync command is invoked on the remote system over ssh (also overriding $PATH, but on the remote system, no perl module involved). Does that help? -Lee Show quoted text
