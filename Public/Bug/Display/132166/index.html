Attached is a patch that updates Makefile.PL to work the same as Crypt::OpenSSL::Bignum does. This allows, for example, Crypt::OpenSSL::EC to link after compile on Strawberry Perl. Without the patch it compiles but does not find libcrypt during linking. This patch also removes hints/MSWin32.pl and does away the need for hints directory. The hints are now incorporated directly in Makefile.PL. I tested this with CentOS 6, 7, 8, old OpenSUSE, MacOS and various Strawberry versions down to 5.14.1.1. Thanks, Heikki -- Heikki Vatiainen <hvn@open.com.au>
