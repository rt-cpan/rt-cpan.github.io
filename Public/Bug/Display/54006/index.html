Subject: Documentation/patch for difference versus Net::XMPP
Hi, The Net::XMPP3 module description mentions: This is a little modified version of Net::XMPP, with fixed one bug, for correctly works with ejabber servers. I do this, because author of Net::XMPP not available from 2007 year. I couldn't see any open issue reported in the Net::XMPP queue, and a diff of Net::XMPP3 1.02 versus Net::XMPP 1.02 doesn't highlight any obvious difference aside from change of author and removal of the CHANGES and STATUS info from the original Net::XMPP. If there is an issue with ejabber it would be helpful to post the information and a patch in the Net::XMPP queue, rather than forking a whole new version. It would also be polite to keep the original CHANGES information and note where your version differs, if at all. Cheers, Stephen
