text/plain 659b
The attached patch makes the setup of %ENV in upload.t and uploadInfo.t work on VMS. There is nothing VMS-specific about my changes so they should work everywhere. Without this I get: $ perl [-.lib.cgi.t]uploadinfo.t Can't make list assignment to %ENV on this system at [-.lib.cgi.t]uploadinfo.t line 32. # Looks like your test died before it could output anything. The patch also removes the entries from %ENV in an END block. Without this, on VMS the changes to the environment persist and are visible after Perl exits. This patch is against blead but I think things are currently in synch enough that you should be able to apply it with patch -p2.
