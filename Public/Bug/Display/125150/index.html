Subject: Fails to build with subversion 1.10.0
After upgrading subversion from 1.9.7 to 1.10.0, SVK fails to build with error in Makefile.PL: ===> Warning: SVN::Core missing or outdated. Please manually install SVN::Core by compiling Subversion (version 1.3.0 or above) with SWIG/Perl bindings, and try this installation process again. The reason is that the check for subversion version does a numerial comparison 1.10.0 >= 1.3.0 that is evaluated as 1.1 >= 1.3.
