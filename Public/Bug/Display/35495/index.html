Hi, The 'more' constructor argument expects an array of simple arguments to pass to the ssh command. In your code, you have to use more => [ '-i', '/home/lloy0076/.ssh/id_rsa' ] I know this is confussing, it's even on the FAQ, but it is required in order to handle file names with spaces and other special character inside.
