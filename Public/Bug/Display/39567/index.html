Subject: Feed get_registered_template_bundle fails w/o bundle id
The ternary check in get_registered_template_bundle does not correctly catch the no-bundle_id instance, so feed.getRegisteredTemplateBundles is never called. It might also be worthwhile to point out in the documentation that the bundle code requires a server URI of http://api.new.facebook.com/restserver.php
