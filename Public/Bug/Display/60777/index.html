Hi Daniel, thanks for your contribution to CPAN. Would you please so kind to update your distribution to provide some minimal requirements for packagers? 1) Use suitable versions - 1.02a is not a "parseable" version number. See http://www.dagolden.com/index.php/369/version-numbers-should-be-boring/ for some details. 2) Add a ChangeLog 3) Add some fields to META.yml (from Makefile.PL), e.g. license If you don't know how, Alexandr Ciornii has sent me a nice code snippet, you can use from a make file of my modules (e.g. http://search.cpan.org/dist/SQL-Statement/) You would ease our (package maintainers) lives dramatically. Thanks, Jens
