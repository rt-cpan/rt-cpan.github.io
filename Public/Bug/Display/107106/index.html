Subject: Incorrect tar file name
The tar file name uploaded to PAUSE is incorrect. The latest version of this module is 0.03, but https://metacpan.org/pod/Data::Structure::Compare points to 0.02 as latest, and it does not even show version 0.03. Also, there is an error when trying to report an issue with version 0.03 at https://rt.cpan.org/Dist/Display.html?Name=data_structure_compare The error message is: "The distribution you specified, data_structure_compare, could not be found." Also, when I ran the following command, it installed version 0.02 instead of version 0.03: cpan Data::Structure::Compare I think the file data_structure_compare-0.03.tar.gz should have been named: Data-Structure-Compare-0.03.tar.gz The next upload should be named: Data-Structure-Compare-0.04.tar.gz Lastly, the "Changes" file in 0.03 is missing an entry for version 0.03.
