First let's make sure that Devel::CheckOS DTRT when asked to consider a single OS: $ perl -e 'use Devel::CheckOS qw(os_is);@oses = qw(Linux); print "OS is ".(os_is(@oses) ? "" : "not ")."[".join(", ", @oses)."]\n"' OS is [Linux] $ perl -e 'use Devel::CheckOS qw(os_is);@oses = qw(MSWin32); print "OS is ".(os_is(@oses) ? "" : "not ")."[".join(", ", @oses)."]\n"' OS is not [MSWin32] That looks good. Now let's ask it to consider two OSes, one that matches and one that doesn't, with the matching one either first or last in the list. $ perl -e 'use Devel::CheckOS qw(os_is);@oses = qw(Linux MSWin32); print "OS is ".(os_is(@oses) ? "" : "not ")."[".join(", ", @oses)."]\n"' OS is [Linux, MSWin32] $ perl -e 'use Devel::CheckOS qw(os_is);@oses = qw(MSWin32 Linux); print "OS is ".(os_is(@oses) ? "" : "not ")."[".join(", ", @oses)."]\n"' OS is [MSWin32, Linux] That looks good too. That's with version 1.71. Rejected. No bug found.
