In RT #125956, we concluded that the problem reported in *this* bug ticket were consistent with the changes in sprintf introduced earlier in the perl5 release cycle. Hence, the code built on top of perl5 must change. I am attaching a patch to t/01-basics.t which corrects the test failure and eliminates all but one of the warnings with which the test was displaying when run against blead. Thank you very much. Jim Keenan
