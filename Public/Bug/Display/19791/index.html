Hi Jason, Well, thanks for the fun morning debugging puzzle. It looks like you're trying to re-implement a lot of code that's already been written. You might want to look at the post_all method in WWW::Myspace::Comment, and also at the comment_myspace script that was installed with the distribution (hint: "perldoc comment_myspace"). I've added reference to that script into the docs for the next release since I think it's only mentioned in the README file now. As for the "sample code doesn't seem to work", try this: while ( my $line = <COMMENT> ) { - $newComment = $line; + $newComment .= $line; $newComment =~ s/(\r\n?|\n)//g; # clean up the extra line endings print ( $newComment ); } It prints fine because you've got a print statement in your while loop while the file's being read, and then you print it *again* with your debugging statement (which prints nothing).
