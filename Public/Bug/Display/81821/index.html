Subject: Add support for EsoAPI

text/plain 519b
There is an extension to Brainfuck (as well as other esoterical languages) called EsoAPI (mentioned http://esolangs.org/wiki/EsoAPI, defined http://kidsquid.99k.org/programs/esoapi/esoapi.html, example implemention http://esolangs.org/wiki/User:JayCampbell/weave.rb). I've attached a patch to add support for this API to the Acme::Brainfuck module. You can test via the attache esoapi-hello.pl. Before the patch it will print "EsoAPI required\n", and after the patch will print "Hello World!\n" Cheers, Hugh
