Before: $ time perl -e 'system "perl -Mblib t/01-basic.t >/dev/null" for 1..10' real 0m1.425s user 0m1.122s sys 0m0.255s After: $ time perl -e 'system "perl -Mblib t/01-basic.t >/dev/null" for 1..10' real 0m1.392s user 0m1.088s sys 0m0.256s The reason this speeds things up is that File::Spec::Unix::canonpath contains a whole list of substitutions: $path =~ s|/{2,}|/|g; # xx////xx -> xx/xx $path =~ s{(?:/\.)+(?:/|\z)}{/}g; # xx/././xx -> xx/xx $path =~ s|^(?:\./)+||s unless $path eq "./"; # ./xx -> xx $path =~ s|^/(?:\.\./)+|/|; # /../../xx -> xx $path =~ s|^/\.\.$|/|; # /.. -> / $path =~ s|/\z|| unless $path eq "/"; # xx/ -> xx Any substitution that fails will leave $path as an object, resulting in many calls to ->stringify.
