Hi. Your module conflicts namespaces with the 'NTLM' distribution on CPAN, which was the first user of the 'Authen::NTLM' namespace. Therefore, it is impossible to have both of these packages working in the same perl environment at the same time, and also breaks automatic installation via CPAN or CPANPLUS. We're looking to support NTLM authentication in Catalyst::Authentication::Credential::HTTP, by using your module, however we can't release this until the namespace conflict is resolved in some way. Is it possible for you to ship a non-conflicting distribution, or, if you no longer have time or inclination to maintain this code, then assign co-maint to me, so that the code can be maintained and used sensibly in other projects? Thanks in advance Tom
