Class::MethodMaker::Engine (and possibly other files in the distribution) has a "use 5.005_62;" statement. But it uses features from 5.006. (use warnings, our, etc.) Please change "use 5.005_62;" to the appropriate version of perl. Also, "use 5.6;" didn't work on pre-5.6, so "use 5.006;" would be the way to express that which gives users of pre-5.6 perls a helpful error message. Thanks.
