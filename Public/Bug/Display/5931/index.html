Hi, When running the sample code (see bottom), the following error occurs: Can't call method "c_get" on an undefined value at /usr/local/lib/perl5/site_perl/5.6.1/alpha-dec_osf/BerkeleyDB.pm line 1179. dist: BerkeleyDB 0.25 perl version: v5.6.1 built for alpha-dec_osf OS: OSF1 psbdeva2 V4.0 1229 alpha I noticed that in the CLEAR implementation, a cursor is freed.
