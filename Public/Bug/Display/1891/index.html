Subject: Warnings for increment and decrement on strings.
Hi, I have been confused with the magic increment of strings in Perl 5.8.0 under 'Linux linux 2.4.19-4GB #1 Fri Sep 13 13:14:56 UTC 2002 i686 unknown'. After some playing around I convinced myself that everything is according to documentation.However it might be nice to have some more warnings for increment and decrement. bernhard@linux:~> perl -w -e 'my $a = "a1"; my $b = $a; $b = $b-1; print "$a: $b\n"' Argument "a1" isn't numeric in subtraction (-) at -e line 1. a1: -1 bernhard@linux:~/devel/parrot> perl -w -e 'my $a = "a1"; my $b = $a; $b--; print "$a: $b\n"' a1: -1 I would have expected the warning message in both cases, as '$b' is incremented by 1 in both cases. A warning messages is especially useful in the case of increment: bernhard@linux:~> perl -w -e 'my $a = "a1a"; my $b = $a; $b = $b + 1; print "$a: $b\n"' Argument "a1a" isn't numeric in addition (+) at -e line 1. a1a: 1 bernhard@linux:~> perl -w -e 'my $a = "a1a"; my $b = $a; $b++; print "$a: $b\n"' a1a: 1 An unsuspecting user might have expected to get 'a1b' instead of '1'.
