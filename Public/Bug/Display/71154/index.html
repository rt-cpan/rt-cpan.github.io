  Thu Sep 22 13:20:59 2011 tomzo [...] users.sourceforge.net - Correspondence added
The first error message you quote indicates that system header file "/usr/include/rpcsvc/rquota.h" is missing on your system. This header is normally part of the standard C library, or possibly an extended development version thereof. On Debian you'd simply enter "apt-get install libc6-dev". All the other errors are just a consequence of the missing header file. If the above doesn't help, please try to help yourself via google or similar. This issue isn't really related to the Quota module, as this system header file is an external prerequisite. (However, if everything else fails, you could fall back to the copy of rquota.h in the include/ sub-directory within the Quota distribution. You'd just have to replace <rpcsvc/rquota.h> with "include/rquota.h" in hints/linux.h. I don't recommend to do this except as a last resort.) -tom
