All (recent ?) versions of libwww-perl include the t/live/jigsaw-chunk.t test script. This relies on an external http server providing http://jigsaw.w3.org/HTTP/ChunkedScript Unfortunately the external server occasionally serves incorrect content, and the test fails. I've had a response from the Jigsaw people to my bug report, so I think the solution is probably to delete the test from libwww-perl. Show quoted text
