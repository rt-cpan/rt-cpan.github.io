The canned default list of allowed pragmata has aged out a bit, and in particular doesn't allow for the C<use 5.xxx> statement used to enable features in perl >= 5.010. Patch updates the default list of allowed pragmata to include this and the C<feature> pragma. While I'm here, the patch also picks up your TODO and add trivial changes to make the list of allowed pragmata configurable.
