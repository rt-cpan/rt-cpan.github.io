Subject: EU::MM 6.74 fails on < 5.8.9 under certain conditions
The manifestation on a freshly compiled 5.8.x is that cpan ExtUtils::MakeMaker works, while cpanm ExtUtils::MakeMaker fails. I did not trace why there is a difference, but the underlying problem is clearly here: https://metacpan.org/source/BINGOS/ExtUtils-MakeMaker-6.74/lib/ExtUtils/Command/MM.pm#L17 ::stat() was introuced in Time::HiRes 1.92 (https://metacpan.org/changes/distribution/Time-HiRes#L233) which did not ship in core until 5.10.0/5.8.9. You need to augment the eval{} with a VERSION call as well (or alternatively just check for ->can('stat') ) Cheers
