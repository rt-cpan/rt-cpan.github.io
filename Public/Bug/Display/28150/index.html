I got the same test failure that srezic reported in May. <http://www.nntp.perl.org/group/perl.cpan.testers/2007/05/msg486124.html> I could determine the cause. It is the locale; I, too, run under de_DE.UTF-8. Setting LANG=C makes pty.t pass cleanly in I::R 0.80 and 0.80_91. The attached logs were generated thus: make test TEST_FILES=t/pty.t TEST_VERBOSE=1 IPCRUNDEBUG=4 > pty.log 2>&1 LANG=C make test TEST_FILES=t/pty.t TEST_VERBOSE=1 IPCRUNDEBUG=4 > pty.LANG-C.log 2>&1 make test TEST_FILES=t/pty.t TEST_VERBOSE=1 IPCRUNDEBUG=4 > pty_91.log 2>&1 LANG=C make test TEST_FILES=t/pty.t TEST_VERBOSE=1 IPCRUNDEBUG=4 > pty_91.LANG-C.log 2>&1
