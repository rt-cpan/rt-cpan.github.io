  Wed Dec 21 04:57:34 2005 TEEJAY [...] cpan.org - Ticket created
Attached is a patch against SVN, which closes a number of possible XSS holes where data is displayed directly to the browser from the database. In the case where one user can see another's data input into a database, you run the risk of having sessions stolen or other badness. Short version: if it didn't come from a trusted place, you better push all strings through the TT "html" filter before displaying it to a browser.
