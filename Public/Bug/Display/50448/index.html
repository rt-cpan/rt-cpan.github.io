Subject: Ignore undef values in @data
If @data contains undef values, then the statistics returned are incorrect. This patch contains changes to min, max, mean, mode, median, sum and count to ignore undef values. The patch also contains test cases to prove the changes were correct. If this patch is accepted, please credit Opsera Limited for the fixes. Thanks for a nice module!
