Hi, Apache::ConfigFile 1.18 doesn't handle line continuations (denoted by a backslash at the end of the line). I'm attaching a patch by Gunnar Wolf that fixes this. It is included in the Debian version of Apache::ConfigFile. (For reference, this was originally reported as Debian bug #309721, http://bugs.debian.org/309721 .) -- Niko Tyni ntyni@iki.fi
