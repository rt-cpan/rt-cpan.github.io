Subject: Server 2003 & Vista Patch
Hi Lee, Thanks for the very useful HTTP::BrowserDetect. I've attached a patch that will provide support for Windows Server 2003 and Windows Vista. Below are the user agents the patch was based on. I've tested on Server 2003 but not on Vista. However, the user agent string for Vista comes from msdn.com and microsoft.com, and is finalized. Windows 2003 IE 6 ================= Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.2; SV1; .NET CLR 1.1.4322) Windows Vista IE 7 ================== According to http://blogs.msdn.com/ie/archive/2006/09/20/763891.aspx http://msdn.microsoft.com/workshop/author/dhtml/overview/aboutuseragent.asp Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 6.0) The one small caveat is Windows XP x64 Edition has the same user agent as Windows Server 2003, not Windows XP (i.e NT 5.2 not NT 5.1). In my opinion as Windows XP x64 Edition is relatively rare compared to XP or 2003, it is better to label the latter two correctly. Is there a chance you will update CPAN with this patch?
