Subject: 172.16.0.0 - 172.16.255.255
From: Joel Peterson
I'd like to add to Anton's notes that the 172.16.0.0 - 172.16.255.255 range is shared by Taiwan and the Netherlands, when in fact it is a private IP range as defined by RFC 1918 (http://www.faqs.org/rfcs/rfc1918.html). I've updated my own ipcountry.dat database, but it would be nice to see Geo:IPfree get updated as the incorrect information cannot be retroactively fixed without a lot of tedious work.
