Subject: Support for relationship selectors (>, +, *)
HTML::Query doesn't currently have support for various CSS relationship selector modifiers. These include ">" (immediate descendent or child), "*" (non-immediate descendent or grandchild) and "+" (immediate sibling). The attached patches add support for these along with tests.
