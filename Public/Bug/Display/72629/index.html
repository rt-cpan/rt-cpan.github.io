Subject: add getline method to iterators
I'd really like to be able to pass iterators back to a PSGI server. wrapping up a DBIC resultset in an ifilter would be a great way to stream data. PSGI relies on the getline method to stream data, which has the same effect as the diamond operator ('<>'). This patch adds the getline method to further duck type the file handle interface.
