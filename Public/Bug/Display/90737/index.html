Well , I had to do some MsWord research; I didn't know that the feature existed (resetting page numbers at a new section). By reverse engineering the generated HTML, it appears that there is a "mso-page-numbers" property that can be set in the section style. So I'll add this as a new option for the "create_section" method. Expect a new release within a couple of days. Cheers, Laurent Dami
