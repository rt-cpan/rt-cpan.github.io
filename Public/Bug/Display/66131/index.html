Subject: redirect old META-spec.html to CPAN::Meta::Spec
The old META spec 1.4 is hosted here: http://module-build.sourceforge.net/META-spec.html This page is outdated as the META spec 2.0 is now published in CPAN::Meta::Spec. However, many documents (in particular POD pages of CPAN modules) still point to this address as the reference specification.
