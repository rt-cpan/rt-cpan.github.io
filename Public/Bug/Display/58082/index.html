Subject: Dependency removed from the CPAN
This module depends on Net::Google::GData, but that module has been removed from the CPAN, making it impossible to install WWW::Blogger. Installation is possible if you manually install Net::Google::GData from http://backpan.perl.org/authors/id/A/AY/AYOUNG/Net- Google-GData-0.01.tar.gz, but otherwise, there's not much one can do. I've emailed the author of Net::Google::GData to figure out what's going on. Oh, and more documentation on how to get WWW::Blogger to work would be useful, include at least a full example :) Cheers, Ovid
