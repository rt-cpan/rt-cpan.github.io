Subject: new task status "not applicable"
I propose a new status / return value for tasks on the taskListPage. Currently we have "done_photo" if the sub returns non-zero and "error_photo" if the sub returns zero. I propose to add an "na_photo" if the sub returns undef. This is convenient for platform-dependent tasks; on the wrong platform we can visually indicate that the task was not run, and on the right platform we can indicate success/failure as usual. -- - - Martin 'Kingpin' Thurn
