Subject: 50cursor.t: SYS_REFCURSOR not defined
Like in RT #80486, in Oracle8, SYS_REFCURSOR is not defined. Instead of CREATE/DROP PROCEDURE, I'd use an anonymous block - see attached patch. This works in Oracle8 and the tests pass. And the test still shows the problem reported in RT #82663 with DBD::Oracle 1.54.
