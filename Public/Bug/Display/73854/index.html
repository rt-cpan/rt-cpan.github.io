No bugs. The module works really well and simplifies a ton of stuff for me. Thank you! Here is a suggested patch to enhance Mail::Outlook. I use it because I often have to send HTML messages. This patch adds HTML capability. Call ->BodyAsHTML with any "true" value and the Body of the message is flagged as being HTML. This is similar to what Powershell offers with the Send-MailMessage cmdlet.
