It appears that recently (around 23 October 2007) TIAA-CREF switched from using the GET form submission method to the POST method. As a result, the code in Tiaacref.pm no longer works (Finance::Quote version 1.13). In other words, Tiaacref can no longer be used as a quote source. Here's the diff for the change that I made to get this working with Gnucash. Note that I'm not convinced that this is a general solution (in particular, I don't know if $cntt is ever non-zero at line 294; I hope not because I don't see where $urlt/$TIAA_URL are defined).
