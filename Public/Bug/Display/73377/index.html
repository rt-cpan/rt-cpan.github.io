Hello! An example script from AnyEvent::SNMP documentation does not work with latest Net::SNMP v6.0.1. It works with Net::SNMP v5.2.0, but produces a lot of warnings like Use of :locked is deprecated at /home/sp/perl5/perlbrew/perls/perl-5.12.4/lib/site_perl/5.12.4/Net/SNMP.pm line 19 with perl 5.12.4 and 5.14.1 (not tested with 5.10.1) The script (with few corrections): ================================================================ use strict; use warnings; use AnyEvent::SNMP; use Net::SNMP; # just use Net::SNMP and AnyEvent as you like: # use a condvar to transfer results, this is # just an example, you can use a naked callback as well. my $cv = AE::cv; # ... start non-blocking snmp request(s)... Net::SNMP->session(-hostname => "10.99.99.41", -community => "xxxxxxxxxx", -nonblocking => 1)->get_request(-callback => sub { $cv->send(@_); }, -varbindlist => ['1.3.6.1.2.1.1.5.0']); # ... do something else until the result is required my @result = $cv->wait; EV::loop(); ================================================================ With Net::SNMP 6.0.1: $ perl -w x.pl Can't locate object method "msg_handle_alloc" via package "AnyEvent::SNMP" at /usr/lib64/perl5/site_perl/5.10.1/Net/SNMP.pm line 2200. $ Tested on: 1. Gentoo Base System release 2.0.1 Linux host 2.6.34-gentoo-r1 #1 SMP Mon Aug 16 12:45:00 MSD 2010 x86_64 Intel(R) Core(TM)2 Quad CPU Q9400 @ 2.66GHz GenuineIntel GNU/Linux perl 5.10.1 installed from Gentoo distribution AnyEvent version 6.1 installed from Gentoo distribution. EV installed from Gentoo distribution. Net::SNMP latest, installed using cpan. AnyEvent::SNMP latest, installed using cpan. 2. Debian 6.0.3 amd64 perl 5.14.2 and perl 5.12.4 both installed via perlbrew. EV, AnyEvent, Net::SNMP, AnyEvent::SNMP was installed using cpamn. ====================================================================== I've added to AnyEvent::SNMP.pm the following 3 lines sub msg_handle_alloc { return $MESSAGE_PROCESSING->msg_handle_alloc(); } from Net::SNMP::Dispatcher and script started to work. -- Sergey Potapov.
