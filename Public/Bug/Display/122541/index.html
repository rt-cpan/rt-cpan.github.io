This bug seems to have been introduced in DBD::mysql version 4.038_01 and is still present (4.043), tested on CentOS 6.5 with perl 5.10 and CentOS 7.2 with perl 5.24. Minimum reproducible code: use DBI; my $dbh = DBI->connect("DBI:mysql:host=localhost"); my $stmt = "SELECT round( degrees(0.00043)*69, 2)"; print $dbh->selectrow_array($stmt); result is “1.7” instead of the expected “1.70” for DBD::mysql versions > 4.038.
