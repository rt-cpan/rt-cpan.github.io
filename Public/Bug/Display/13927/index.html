  Sat Jul 30 15:29:31 2005 jpeacock [...] cpan.org - Correspondence added
I'm sorry, the functionality of the CPAN release of version.pm is already included in the 5.9.x code so it is not necessary (nor possible) to build it with 5.9.2. In addition, all release of 5.9.x prior to 5.9.3 (not out yet) contain incompatible implementations of version.pm (due to continued design changes). For the next release of version.pm to CPAN, I will configure Build.PL to exclude compilation under any version of Perl 5.9.x. Thanks for the reports. John
