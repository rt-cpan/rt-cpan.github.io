Special characters in Perl formats make PPI::Document::new() fail. The following script defines a simple format: format STDOUT = Ã¤@<<<<<<< 'Name' . write STDOUT; When trying to build a PPI document for this script by use PPI; use PPI::Dumper; my $module=new PPI::Document('format.pl'); my $dumper=PPI::Dumper->new($module) or die PPI::Document->errstr; the constructor fails with Fatal error... regex failed to match in 'Ã¤@<<<<<<< ' when expected at /.../site_perl/5.10.0/PPI/Token/Word.pm line 178.
