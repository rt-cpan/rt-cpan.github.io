[Cause] Test::EOL::_all_files() filters out Module::Install stuff, that is, files under inc/ directory. But current Test::EOL::_all_files() filters out not only inc/ directory but also *inc*/ directory. I make Test::EOL module on /home/tzccinct/... directory which includes a word "inc" in the middle of my name. So all the files under /home/tzccinct directory are filtered out unexpectedly. If you are OK, would you apply the attached patch?
