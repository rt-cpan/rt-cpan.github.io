I got a make test error when I was trying to "install Bundle::CPAN" today. The error is: "Can't locate TAP/Parser/Source/Perl.pm" I @INC (list of @INC) at /usr/lib/perl5/5.8.8/TestHarness.pm line 12. I tried to install via "perl -MCPAN -e shell" and then downloaded the module and build manually. Both had the same problem. I think the problem is the Perl.pm module is under "TAP/Parser/SourceHandler/Perl.pm". As a workaround I created I "/usr/lib/perl5/5.8.8/TAP/Parser/Source" and copied /usr/lib/perl5/5.8.8/TAP/Parser/SourceHandler/* to /usr/lib/perl5/5.8.8/TAP/Parser/Source. The build then proceeded OK.
