Subject: Please sync with bleadperl
In the course of checking if bleadperl is currently in sync with CPAN modules I have found that Carp is out of sync: A single line is missing from lib/Carp/Heavy.pm in bleadperl, namely: $VERSION = eval $VERSION; (immediately underneath "our $VERSION = '1.35';"). I considered simply inserting the line into blead's copy to make them in sync but then 5.21.10's Heavy.pm would differ from 5.21.9's Heavy.pm, so I would need to bump the $VERSION... which would then make blead out of sync with CPAN again anyway. Therefore, could you please make a new CPAN release with no changes other than a $VERSION bump to 1.36? I can then bring that into bleadperl and we're back in sync again. (Alternatively, delete the extra line from the CPAN release (as well as bumping the $VERSION, obviously) if in fact the blead version is correct.) It would be great to have this before Friday 20th when 5.21.10 is released, but I dare say such a minor thing as a $VERSION bump would be allowed in afterwards if need be.
