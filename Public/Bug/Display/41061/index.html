This is not a Mail::SPF issue but a general SPF issue, so the Mail::SPF bug tracker is not the right place to discuss it. Unless you have evidence that this is a problem with Mail::SPF specifically, please do not reply to this message. However, for the record: The SPF spec does not accept trailing dots on the "all" mechanism. This would be a syntax error. Please check the SPF spec (RFC 4408).
