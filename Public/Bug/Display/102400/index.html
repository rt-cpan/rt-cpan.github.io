I have added support for the set_callback method but in a different way, taking advantage of the *_user_data methods provided by the library to store the callback pointer, the self reference and any custom user data. Also, I have changed the meaning of the callback return values. Now, in my implementation, the callback must return a true value to keep the training going on or false to stop it. You can get the development version from GitHub: https://github.com/salva/p5-AI-FANN I have barely tested the new features so, use with care and please, report any problems you may find. Cheers
