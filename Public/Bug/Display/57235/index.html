Subject: Unclear dependency for XML::SAX::Base
Currently Expat depends on XML::SAX::Base Unfortunatley, 2 CPAN modules install this pm file. XML::SAX - http://search.cpan.org/~grantm/XML-SAX-0.96/ XML::SAX::Base - http://search.cpan.org/~khampton/XML-SAX-Base-1.02/ Problem 1: XML::SAX::Base is listed as unautorized in CPAN search, even though it comes up as the provider in 02packages.details.txt.gz Problem2: XML::SAX provides a lib/XML/SAX/Base.pm, which if installed later would overwrite XML::SAX::Base's lib file. The owners of Expat seem to mostly own the other 2 and since Expat is the depender, I thought this RT would be the place to sort this out. Right now it really confuses things when packaging them up as RPMs. Thanks,
