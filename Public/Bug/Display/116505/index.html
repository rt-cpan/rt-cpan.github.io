Subject: CVE-2016-1238: ptar, ptardiff, ptargrep may load optional modules from the current directory
The attached patch prevents ptar, ptardiff and ptargrep from loading optional compression modules from the default current directory entry in @INC. This change is included in maint-5.22, maint-5.24 and blead perl. Tony
