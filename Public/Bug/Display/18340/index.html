Good morning. I'm hoping to get away from maintaining POE's .ppd files. The current version (pre-0.34) has a dumbed down installer that should be more compatible with ActiveState's package builder. I'm hoping that they'll build better .ppd files than I have been---and that Windows users can just "install POE" through ppm without going through repository gyrations. Because we're dealing with a third-party package build system, it may take some time (and minor releases) to work out all the installer problems. Wish us luck!
