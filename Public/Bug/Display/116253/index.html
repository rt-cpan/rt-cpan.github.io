Subject: YAPC::Europe is not a module
In two modules, Acme::CPANLists::Import::DatesPart2 and Acme::CPANLists::Import::RomanNumbers, you mention a "YAPC::Europe" module. This module does not exists. You should remove it from the list or display it in a "not really a module" paragraph. Same thing about "next::method" in the ...::ORM module and "site::header" in the ...::TemplatingSystem module. And maybe others as well. Or rather, you should add a stop-list to the generating module and change the generation process to remove these pseudo-modules or tag them as non-existing.
