Subject: patch to callback problem
Currently Gimp module don't return any return values from Perl-Fu. I think I've solved the problem. At least the patch solved my problem for now. :) If you don't see attached patch, it's also at http://freebsd.sinica.edu.tw/~plasma/gimp-return_value.patch. How To Verify Apply the patch file, then install Gimp module. A tiny Perl-Fu script is placed at http://freebsd.sinica.edu.tw/~plasma/tiny-test.pl. Put it in ~/.gimp/plug-ins, then start Gimp. Invoke a perl debugger with 'perl -de 0', then type the following commands: use Gimp ":auto" use Gimp::Fu Gimp::init x perl_fu_tiny_test("Text","Charter",45) Below is my output: DB<66> x perl_fu_tiny_test("Text","Charter",45) 0 Gimp::Image=SCALAR(0x85bb0e4) -> 0 1 Gimp::Layer=SCALAR(0x85b7384) -> 2 Now it really return values. :) Best Regards, Chen, Wei-Hon
