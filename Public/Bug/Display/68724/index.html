Hello Serguei, Thanks for this plugin! I'm considering removing the dump() and dump_html() methods from a future version of CGI::Application, and instead having this functionality handled by a DevPopup plugin. This plugin looked like the logical place to put it. Would you be willing to add displaying the current runmode and environment details to this plugin?
