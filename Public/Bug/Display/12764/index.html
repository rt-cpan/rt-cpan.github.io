Subject: Cygwin: CPANPLUS cannot send test reports but Test::Reporters cpantest utility can
I get the following error for sending a report from within cpanp shell: [MSG] DEFAULT 'munge_test_report' HANDLER RETURNING 'sub return value' [ERROR] Could not send 'pass' report for 'MailTools-1.67': Test::Reporter: Unable to send test report at /usr/lib/perl5/site_perl/5.8/CPANPLUS/Dist/MM.pm line 577 [ERROR] Failed to send test report for 'Mail::Send' at /usr/lib/perl5/site_perl/5.8/CPANPLUS/Module.pm line 585 Yet the Test::Reporter 1.27 cpantest tool works fine, and the native Win32 CPANPLUS is able to send reports, so I'm pretty sure it's not a firewall issue.
