Hi, $rfc->sapinfo() actually returns a hash ref, so you need to dereference it so that you can get at the data: $test = $rfc->sapinfo() or die "I don't know"; map { print "key: $_ = ", $test->{$_}, "\n" } sort keys %$test; Cheers.
