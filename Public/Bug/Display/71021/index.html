Subject: Added --nohtmlheader option to Pod::Html and pod2html
Download (untitled) / with headers
text/plain 263b
Currently pod2html and Pod::Html always generate HTML headers and footers. This is annoying when trying to embed the output in some other HTML document. The attached patches introduce the --nohtmlheader option which allows you to omit the HTML header and footer.
