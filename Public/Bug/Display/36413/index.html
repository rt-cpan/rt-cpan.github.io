Subject: Daemon::Generic as init script not LSB conformant

text/plain 452b
Daemon::Generic has some init script semantics, but is not LSB conformant which for example makes it unusable in combination with heartbeat. The attached patch tries to improve the situation by providing a status action and correct exit codes. It also cleans up the pid file on stopping the daemon. It is incompatible with the current version in so far as the exit code of the start action got changed to 0 in case the daemon is already running.
