> is there any place we can predefine PREFIX and ftp and http gateways.
For gsteways we have the {ftp,http,no}_proxy variables, for PREFIX we have make_install_arg variable. To determine where things go wrong, it is necessary to find out how connections to the outside world do work without CPAN.pm. Only then we can determine if CPAN.pm supports this setup of your firewall. Please retry with a recent version and reopen this bug ticket if you still have problems and please support some concise information how your firewall works and how you go through it *without* CPAN.pm. Thanks!
