XML::SAX v0.96 included XML::SAX::Base v1.04. XML::SAX::Base was removed from XML::SAX v0.99 but XML::SAX still has a dependency on XML::SAX::Base v1.05. Unfortunately, the most recent XML::SAX::Base on CPAN is v1.02 provided by Kip Hampton. If XML::SAX v0.99 will work with XML::SAX::Base v1.02, please can the dependency information in XML::SAX be updated. If XML::SAX genuinely requires XML::SAX::Base v1.05 then please can you work with Kip to update his release or include XML::SAX::Base in XML::SAX again. Many thanks.
