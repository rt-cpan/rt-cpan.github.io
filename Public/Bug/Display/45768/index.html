META.yml contains a "resources: Repository:" as per META spec 1.4. However, I've noticed that search.cpan.org picks up "resources: repository:" (note case), and inserts it into the distribution page. For an example, see the link to the source repository for Test-Simple [1], as pointed to in Test-Simple's META.yml [2].
