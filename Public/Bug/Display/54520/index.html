Subject: -.*ButtonAction args not passed to add.*Page methods
When calling any /add.*Page/ method (e.g. addMultipleChoicePage) all the arguments are passed on to the private page-creation subroutine (e.g. _page_multiple_choice). The /-.*ButtonAction/ arguments (e.g. -preNextButtonAction) arguments need to be passed instead to the addPage subroutine.
