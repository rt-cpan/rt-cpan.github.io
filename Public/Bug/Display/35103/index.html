I plan to use Petal for a new project in our university. We are very much impressed with this Module. But while testing it, I run over an unexplicable situation: If I have text in a template source which looks like a perl variable name: $varname this text will disapear when I execute the Petal process method. It does not disapear, if I use a tal:content="text_with_dollars" attribute to include it. It does not disapear, if it is just a one character name '$n' It does not disapear, if it is a number '$10' But this one will disapear: '$nn' Am I missing something?
