Subject: Option to use a HTTP proxy
We need to access the validation service through a HTTP proxy. I guess that's a need other people might add. Hopefully you'll consider adding this to the released module. Since the LWP::UserAgent isn't exposed through the API, we've modified the W3C object to have another attribute: proxy which is applied to the UserAgent object within the code. A patch file is attached for application to version 0.11. Seems to work OK on perl-5.8.7. Thanks for the useful module. Carl Carl Vincent Systems Manager Netskills www.netskills.ac.uk
