Subject: Can't diff strings with "wide" characters because of Digest::MD5
Download (untitled) / with headers
text/plain 294b
A report on londom.pm could not use Test::XML on UTF-8 strings. This is because XML::SemanticDiff uses Digest::MD5 and that module does not like multi-byte characters. There is a work-around in the Digest::MD5 documentation that I have applied to XML::SemanticDiff. See attached patch. Robin
