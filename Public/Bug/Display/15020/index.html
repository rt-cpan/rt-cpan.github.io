Subject: deep_search_distinct_where() Addition (Supports Distinct Queries)
I wanted to use deep_search_where() but retrieve only distinct entries. While I noticed the "DISTINCT" directive in deep_search_where() it only does it for all fields. I wanted to do it for a particular field. Please take a look, and hopefully you will agree to add it! (See Patch) Mark Rorabaugh (2RAD.net)
