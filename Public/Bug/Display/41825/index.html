Hello, we like to use the Net::uFTP module to migrate connections from unsecure FTP to secure SFTP. There are server site some listener not equal to standard port (FTP: 21, SFTP: 22), for example 10021. One Perl-Prozess mangages SFTP and FTP connections the same time with Net::uFTP in future. Is ist possible to add this option (for example: port => $port) to the new constructor? The same problem we have with option (Timeout => xx).
