Subject: Makefile.PL should not always prompt for user input
Hi, Attached is a patch that adds a flag to Makefile.PL to allow the package builder to enable or disable the live tests. This lets package builders build the package without having it prompt for user input. We're using this patch on Fedora for that purpose. Thanks, -RN
