thanks for this report too! from the gdb backtrace (2^10 thanks), the problem appears to be related to libjemalloc, which the upstream c++ package uses by default. I've updated Build.PL to add the '--disable-jemalloc' option by default for all alien builds. if jemalloc is indeed the culprit, this bug should be fixed in Alien-DDC-Condcordance v0.005.
