Subject: Memory leaks at Load()
There is memory leaks at YAML::XS::Load() method. Tested on 2 platforms: FreeBSD 8.0-CURRENT #8: Sat Jan 17 19:42:04 MSK 2009 i386 FreeBSD 7.2-PRERELEASE #6: Mon Mar 23 23:09:10 MSK 2009 amd64 Attached test eats 1GB at i386
Subject: t_yaml_leak.pl
#!/usr/bin/perl use strict; use warnings; use YAML::XS; my $count = 1_000_000; my $yml = <<'TXT'; --- session_code: 649e116d890ad16306ef8d4491029db9 country: xx exchrate: 33.81 id: 14695373 message: xxxxx number: 1234 operator: xxx operator_full: xxx phone: 7701xxxxxxx plain: xxxxx price: 31.56037 TXT my $t = time(); foreach (1..$count) { my $y = YAML::XS::Load($yml); } $t = time() - $t; my $s = int($count/$t); print "YAML::XS::Load: time: $t sec; speed: $s req/sec\n";
