> List::Util version 1.42 required--this is only version 1.38 at /usr/local/lib/x86_64-linux-gnu/perl/5.20.1/Sub/Util.pm line 22.
It looks like you have an older version of List::Util installed earlier in @INC, which breaks because the XS parts of Sub::Util are actually implemented in List::Util. You can see where it's installed by running perl -MList::Util -E 'say $INC{"List/Util.pm"} Or, if you want to see all the versions in @INC, install the 'V' module and run: perl -MV=List::Util -- "I use RMS as a guide in the same way that a boat captain would use a lighthouse. It's good to know where it is, but you generally don't want to find yourself in the same spot." - Tollef Fog Heen
