Subject: Small mistakes in the documentation
Hello Jos, I found a few minor mistakes in the documentation. - CPANPLUS::Module::Author has =head1 NAME CPANPLUS::Module, which masks the documentation of CPANPLUS::Module when looking the documentation throught Search CPAN. - CPANPLUS::Shell::Default has =head1 NAME CPANPLUS::Shell::Default; (note the ";"), thus not linking the documentation to the module in Search CPAN. - at the end of the documentation of CPANPLUS::Configure, in "See Also", there is a link towards "CPANPLUS::Conf", which does not exist. Also, one can see comments used to configure your editor :-) - CPANPLUS::Module::Checksums is missing the traditionnal NAME, SYNOPSIS, DESCRIPTION, METHODS sections - some Pod formatting errors in CPANPLUS::Hacking I'll try to send you a patch which correct some of these later.
