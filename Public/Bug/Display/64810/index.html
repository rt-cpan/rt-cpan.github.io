  Tue Jan 18 02:06:20 2011 kmx [...] cpan.org - Correspondence added
Hi Alex,

the trouble is that Alien::SDL on MS Windows platform is using pre-built binaries which are currently available only for 32bit/x86 and 64bit/x64 gcc compiler (e.g. for strawberry perl) - you seem to have Microsoft compiler ('cl') therefore Alien::SDL is not able to offer you a valid install option.

Currently there are basically 2 ways how to handle your issue:

1/ preparing SDL's + all related libs' binaries for MSVC compiler (which requires significant effort) if you wanna volunteer you can have a look at gcc binaries
-  http://strawberryperl.com/package/kmx/sdl/Win32_SDL-1.2.14-extended-bin_20100704.zip
- and my build script 06-build-SDL-all.bat in http://svn.ali.as/cpan/users/kmx/strawberry_packs_devel/

2/ you may consider using Strawberry Perl
