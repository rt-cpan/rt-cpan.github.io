> The bundle file "\.cpan\Bundle\CPAN.pm" may be a broken > bundlefile. It seems not to contain any bundle definition. > Please check the file and if it is bogus, please delete it. > Sorry for the inconvenience.
I've experienced the same problem. The following change solved these problems for me: In CPAN.pm, line 4942, change $from = $self->find_bundle_file($dist->{'build_dir'},$me); to $from = $self->find_bundle_file($dist->{'build_dir'},$me[-1]); Otherwise find_bundle_file doesn't find the correct bundle file (the array item $me[-1] is the name of the file which should contain the bundle info, whereas the scalar $me is a part of the install path). Hope this helps ! Regards, Thierry
