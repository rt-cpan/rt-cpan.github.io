Hello, I am currently maintaining packages for Parabola GNU/Linux-libre. We are a Free Software Foundation approved distribution. As such, we regularly check the source code to ensure that all of our software is freely distributable and modifiable in compliance with the GPL License. Unfortunately, we had to remove Perl-FCGI due to a compliance issue. If possible please re-license your software to be GPL compatible, so that we will be able to package Perl-FCGI and the various GL-compatible programs that depend on it.
