Subject: make_link should probably not return the empty list
If make_link() does not find a URL (e.g. for <a> tags without a href), then the empty list is returned (see https://metacpan.org/source/CORION/WWW-Mechanize-PhantomJS-0.22/lib/WWW/Mechanize/PhantomJS.pm#L1168 ). However, this breaks https://metacpan.org/source/CORION/WWW-Mechanize-PhantomJS-0.22/lib/WWW/Mechanize/PhantomJS.pm#L1355 --- in this case the \%opt argument is moved to the first position!
