Hi Your file INSTALL.html (http://search.cpan.org/dist/DBD-mysql/lib/DBD/mysql/INSTALL.pod) section "Windows/CygWin" is extremely useful, thank you. Please could you make some updates? 1) It appears that the procedure no longer works in MySQL v5.5. E.g. see this link http://forums.mysql.com/read.php?117,407537,409814#msg-409814 which suggests the cygwin config is no longer tested and seems to be broken. Of course it would be great if someone could persuade the MySQL team to fix this, but in the meantime, please could you update the DBD-mysql documentation to advise using v5.1? 2) As per this MySQL article (http://forums.mysql.com/read.php?51,389833,389833), I am using this command ./configure --prefix=/usr/local/mysql --without-server --without-readline CFLAGS=-O2 3) The download link has changed, try http://dev.mysql.com/downloads/mysql/ Many thanks Adam
