Subject: Wrong links in documentation
It seems that links in all mod_perl2-related Pod pages are broken. It is visible in Pod viewers which can handle links, e.g. tkpod or in search.cpan.org's documentation pages (e.g. http://search.cpan.org/~gozer/mod_perl-2.0.4/docs/api/Apache2/Const.pod). Links are in the form L<Apache2::Access|docs::2.0::api::Apache2::Access>. To correct this, the links should simply be in the form L<Apache2::Access>. Regards, Slaven
