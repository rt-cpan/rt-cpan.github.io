Subject: M::B doesn't write a packlist
Hi, I thought this was reported already, but didn't fid it anymore in RT, not even in the resolved bugs. From the CPANPLUS::Dist::Build TODO: =item * Uninstall modules installed by Module::Build Module::Build doesn't write a so called C<packlist> file, which holds a list of all files installed by a distribution. Without this file we don't know what to remove. Until Module::Build generates this C<packlist>, we are unable to remove any installations done by it.
