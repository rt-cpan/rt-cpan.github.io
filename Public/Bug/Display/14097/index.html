For a really simple program: #!/usr/bin/perl -w use strict; use HTML::Template; my $template = HTML::Template->new('filename'=>"test.tmpl"); print $template->output; And this invalid template: Before. <TMPL_IF NAME="TEST"> 1. <TMPL_ELSE> 2. <TMPL_ELSE> 3. </TMPL_IF> After. HTML::Template does not output any error and just simply outputs the 2. case.
