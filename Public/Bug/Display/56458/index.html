Subject: Alarming errors when a download method fails
Alarming errors when a download method fails, even when the next one works. In Makefile_1.PL and the attached log, log1.txt, see how the FTP download failed, many warnings were displayed, then the HTTP download was attempted and successful. The error in the FTP process lets the user think that M::I::Bundle failed to bundle the specified dependency.
