Subject: Rename module (casing)?
I know this is perhaps late and probably not very important, but most other modules use WebService:: (capital S) instead of Webservice. In fact, your module is the only one on CPAN using the lowercase S :) So, perhaps rename the module to use uppercase S? Just a suggestion. Reference: http://blogs.perl.org/users/steven_haryanto/2014/07/net-vs-www-vs-webservice.html Regards, Steven
