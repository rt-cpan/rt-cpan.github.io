Hello, The implementation may be possible if the tailoring is not too complicated and the appropriate reference is available. though CLDR provides ug support only in the Arabic script. http://unicode.org/cldr/trac/browser/tags/release-29/common/collation/ug.xml (note: recent Unicode::Collate::Locale almostly bases on CLDR 22.1) The sort order difference between Uyghur Cyrillic and DUCET Cyrillic seems to be only that of schwa. Uyghur: ..Ie < Schwa < Zhe... DUCET: A < Schwa < Be... Parhaps it is relatively simple. SADAHIRO Tomoyuki Show quoted text
