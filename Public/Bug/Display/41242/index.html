I still get the following bugs/errors/warnings: 1 - Creating a new app % dzil new My::App ARRAY(0x1a6c330) # I doubt the output "ARRAY(0x1a6c330)" is intentional here. 2 - Building the new app % dzil build|test ... couldn't find a place to insert VERSION section to lib/My/App.pm ... I guess what I'm asking for is a bit more documentation/information on how to configure Dist::Zilla. I'd send a draft in myself but I'm obviously not fully clued up on how to use it. Thank you.
