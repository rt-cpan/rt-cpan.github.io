Hi! FreeBSD ports are now including the latest libopendkim 2.10.3, and v2.10.* removed support for ADSP. The outcome is that Mail::OpenDKIM no longer builds/installs, as the XS file will not compile. Mail::OpenDKIM is marked as deprecated by FreeBSD, which recommends switching to Mail::DKIM - but the latter is too slow for our needs. Is a fix planned? If not, would you be able to accept a patch?
