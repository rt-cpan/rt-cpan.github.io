text/plain 1.2k
Make load_class() return the same error as require() try_load_class() will now only return the actual error, without an " at FOO line BAR." suffix, which would only point to Class::Load's code anyway. load_class() and load_optional_class() now chomp the error returned by try_load_class() before croaking it. Users can now replace require() in their code with load_class() and they will get the same error message on failure. E.g.
