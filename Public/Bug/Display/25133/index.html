Hi, recent changes to the uri filter in Template-Toolkit that converts / to %2F which causes URI:s to be incorrect everywhere. This page has more information about the change: http:// search.cpan.org/~abw/Template-Toolkit-2.18/lib/Template/Filters.pm#uri Reverting back to Template-Toolkit 2.15 solves the problem tho. Thanks for a great SVN browser /Claes
