Hi, I can't seem to make a working File-Scan-ClamAV module when using ClamAV-0.95. It works with ClamAV-0.93, but with the latest ClamAV it always fails make test in the same way. I've tried both the latest (1.91) and with an older (1.8) module and both fail in the same way. I've also tested three different Perls, the one that comes with Solaris 2.11 (5.8.4 compiled wit Sun Workshop compiler) and 5.10.1 compiled with either the Sun Studio compiler and GCC. Same thing which ever, and that would rule out Perl itself. I suspect there has been changes in the API for ClamAV, in which case the File-Scan-ClamAV module also needs an update. See below for info.
