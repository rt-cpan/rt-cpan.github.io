Subject: Patch: enable new Trailingcomma option when available
The latest version of Data::Dumper (currently available only as part of Perl 5.24, but it may at some point get an updated dual-life release) has a new Trailingcomma option that emits an extra comma after the last element of an array or hash that would otherwise end a line. The attached patch enables this option in Data::Dumper::Concise when possible, though everything remains unchanged when run under older versions.
