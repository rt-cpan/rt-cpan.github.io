In the GNU inetutils documentation, the NETRC environment variable exists to override which netrc file is used to obtain authentication credentials. https://www.gnu.org/software/inetutils/manual/html_node/The-_002enetrc-file.html This behavior is missing from Net::Netrc.pm, but is easily added to the module. I’ve prepared a patch using the latest libnet-3.07 as a base and have attached the original, the patched and a diff patch for your consideration for inclusion in an upcoming release of libnet.
