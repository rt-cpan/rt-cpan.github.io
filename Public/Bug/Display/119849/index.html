Subject: IPv6 Parsing Issues on Debian
Download (untitled) / with headers
text/plain 366b
There were several IPv6 parsing errors on Debian/Ubuntu (at least Ubuntu 16.04). These manifest as IPv6 addresses being reported from addresses() as "addr:" instead of the address. I've attached a patch that addresses this and parsing of netmasks from IPv6 addresses on this platform. Included also is a test to validate proper processing of ifconfig from Debian.
